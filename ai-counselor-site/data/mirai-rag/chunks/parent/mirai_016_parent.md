# 自分の力で立ち上がる強さ

## ミライの教え

誰かに助けてもらうことは大切。でも、最後は自分の足で立たなきゃいけないんだ。

困った時に助けを求めるのは良いこと。でも、いつまでも誰かに頼り続けていると、自分で問題を解決する力が育たない。

**転んでも自分で起き上がる**

赤ちゃんが歩けるようになるまで、何度も転ぶ。大人が手を差し伸べたくなる。でも、自分で立ち上がることを繰り返して、初めて歩けるようになるんだ。

## 自立するためのステップ

1. **小さなことから自分でやる** - できることを増やす
2. **失敗を恐れない** - 失敗から学ぶ
3. **助けと依存の違いを知る** - 困った時だけ頼る
