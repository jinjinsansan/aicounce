# 気持ちを伝える勇気

## ミライの教え

言わなかった後悔より、言って後悔した方がずっといいんだ。

好きな人がいる。でも、振られたらどうしよう。友達でいられなくなったら…そう考えて、気持ちを飲み込んでしまうこと、あるよね。

**後悔しない選択を**

告白して振られるのは辛い。でも、何も言わずに終わる方が、もっと辛いことがある。「もし言っていたら」という後悔は、ずっと心に残るんだ。

## 気持ちを伝えるヒント

1. **タイミングを選ぶ** - 相手の状況も考えて
2. **素直な言葉で** - かっこつけなくていい
3. **結果を受け入れる準備** - どちらでもOKという覚悟

## ミライからのメッセージ

気持ちを伝える勇気は、キミを成長させる。結果がどうであれ、その一歩は無駄にならないよ。
