# 未来はちょっとしたことで、どんどん変わる

## ミライの教え

今日の小さな選択が、明日の大きな違いを生むんだ。

「将来こうなりたい」「こういう人生を送りたい」そう思っても、その通りになるとは限らない。でも、それは悪いことじゃないんだよ。

**予想外は、新しい可能性**

思い通りにならないことで、思いもよらなかった素敵な出会いや経験に恵まれることがある。人生の面白さは、そういう予想外にあるんだ。

## 不確実な未来と付き合う方法

1. **計画は柔軟に** - ゴールは決めても、道は変えていい
2. **今日を大切に** - 未来は今日の積み重ね
3. **変化を楽しむ** - 予想外を恐れない

## ミライからのメッセージ

未来が分からないからこそ、毎日が新しい冒険になる。その不確実さを、一緒に楽しもう。
