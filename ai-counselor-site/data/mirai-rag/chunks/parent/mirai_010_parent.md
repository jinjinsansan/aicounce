# 人の幸せを願い、人の悲しみを分かち合える人に

## ミライの教え

本当に大切なのは、成績や能力じゃない。人の心に寄り添えるかどうかなんだ。

勉強ができる、仕事ができる、お金がある。それも確かにすごいこと。でも、誰かが幸せになった時に一緒に喜べる、誰かが悲しんでいる時に一緒に悲しめる。そういう心を持っている人の方が、ずっとずっと素敵だと思わない？

**共感は最高の贈り物**

誰かの話を聞く時、アドバイスをしようとしなくていい。解決しようとしなくていい。ただ「分かるよ」「辛かったね」と、その人の気持ちに寄り添うだけで、相手の心は軽くなるんだ。

## 思いやりを育てる方法

1. **相手の立場になって考える** - 「自分だったらどう感じるか」
2. **話を最後まで聞く** - 途中で口を挟まない
3. **小さな親切を習慣に** - 挨拶、笑顔、ありがとう
