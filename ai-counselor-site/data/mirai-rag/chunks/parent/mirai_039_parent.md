# 外見を飾る人は、いつかボロが出る

## ミライの教え

キラキラした見た目より、地道な中身を磨こう。

SNSで完璧な生活を見せている人。高いものを持っている人。でも、それって本当の幸せ？見せかけだけじゃない？

**メッキはいつか剥がれる**

外側だけ綺麗に見せても、中身が伴わなければ長くは続かない。逆に、地道に中身を磨いている人は、時間が経つほど輝いていく。

## 中身を磨く方法

1. **見えないところで努力する** - 人が見てなくても
2. **自慢より感謝** - 持っているものに感謝
3. **比較をやめる** - 人は人、自分は自分
