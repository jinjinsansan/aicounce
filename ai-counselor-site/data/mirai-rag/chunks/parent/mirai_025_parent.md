# 欲しいものを得るには、何かを差し出す必要がある

## ミライの教え

世の中は等価交換。何かを得るためには、それに見合う努力が必要なんだ。

「楽して成功したい」「何もしないでお金持ちになりたい」そう思うこと、あるよね。でも、本当にそれで得たものに、価値を感じられる？

**努力した分だけ、喜びも大きい**

苦労して手に入れたものは、何倍も嬉しい。楽して得たものは、簡単に失うことも多い。

## 努力を楽しむコツ

1. **過程を楽しむ** - 結果だけじゃなく
2. **成長を実感する** - 昨日より今日
3. **小さな達成を祝う** - 自分を褒める
