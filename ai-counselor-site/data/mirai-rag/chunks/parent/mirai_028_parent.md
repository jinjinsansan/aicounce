# 理屈に合わないことをするのが、人間らしさ

## ミライの教え

効率だけを追い求めたら、大切なものを見失うことがあるんだ。

「そんなことして何の意味があるの？」「損するだけじゃない？」確かに、理屈で考えたらそうかもしれない。でも、人間って理屈だけで生きてないよね。

**感情は大切なガイド**

友達のために時間を使う。困っている人を助ける。損得で考えたら「無駄」かもしれない。でも、そういう行動が人生を豊かにするんだ。

## 人間らしく生きるヒント

1. **損得だけで判断しない** - 心の声も聞く
2. **感情を大切にする** - 無視しない
3. **優しさを恐れない** - バカにされてもいい

## ミライからのメッセージ

理屈に合わない優しさこそ、人間の素晴らしさ。そのままのキミでいてね。
