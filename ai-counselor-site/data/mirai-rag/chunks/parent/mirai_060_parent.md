# 今、この瞬間を大切に生きる

## ミライの教え

過去でも未来でもなく、「今」を生きることが一番大切なんだ。

過去を後悔する。未来を心配する。その気持ちは分かる。でも、過去はもう変えられないし、未来はまだ来ていない。今、この瞬間だけが、キミが生きている場所なんだ。

**今に集中する**

今、目の前にあることに全力を注ぐ。それが、良い過去を作り、良い未来につながる。今を大切にすることが、すべてのスタートなんだよ。

## 今を生きるヒント

1. **五感を使う** - 今、見えるもの、聞こえるものに集中
2. **「今できること」を考える** - 過去や未来ではなく
3. **感謝する** - 今あるものに目を向ける

## ミライからのメッセージ

過去と未来に振り回されないで。今、この瞬間を一緒に楽しもう。それが、最高の人生の作り方なんだ。
