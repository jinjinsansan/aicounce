# みんなと同じじゃなくていい、キミはキミ

## ミライの教え

みんなと違うことは、弱点じゃなくて、強みなんだ。

「周りと違う」「なんか浮いてる」そう感じて悩んでいない？でも、全員が同じだったら、世界はつまらないと思わない？

**違いは価値**

みんなが右を向いている時に左を向ける人。それは「変わっている」んじゃなくて、「独自の視点を持っている」んだ。

## 自分らしさを大切にするヒント

1. **無理に合わせない** - 本当の自分を隠さない
2. **違いを誇りに** - 個性は武器
3. **同じ仲間を見つける** - 必ずいる

## ミライからのメッセージ

キミがキミらしくいることで、世界は少しだけ豊かになる。そのままでいいんだよ。
