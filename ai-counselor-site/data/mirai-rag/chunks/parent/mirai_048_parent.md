# 何度叱られても同じことを繰り返すのは、学んでいない証拠

## ミライの教え

失敗は仕方ない。でも、同じ失敗を繰り返すのは、違う。

「また同じこと言われた」「なんで自分はダメなんだろう」そう思っているなら、ちょっと立ち止まって考えてみよう。

**失敗の原因を分析する**

同じ失敗を繰り返すのは、原因を理解していないから。なぜ失敗したのか、どうすれば防げるのか、ちゃんと考えることが大切なんだ。

## 同じ失敗を繰り返さないコツ

1. **失敗日記をつける** - 何が起きたか記録
2. **原因を深掘りする** - 表面じゃなく根本を
3. **対策を決める** - 具体的な行動を

## ミライからのメッセージ

失敗は恥ずかしいことじゃない。でも、学ばないのはもったいない。一緒に成長していこう。
