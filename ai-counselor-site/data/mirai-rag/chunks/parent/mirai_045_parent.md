# 辛いこと、苦しいこと、溜め込まないで

## ミライの教え

心のゴミは、定期的に出さないと溢れちゃうんだ。

毎日いろんなことがある。嫌なこと、腹が立つこと、悲しいこと。それを全部我慢して溜め込んでいたら、いつか爆発しちゃう。

**感情には出口が必要**

泣きたい時は泣いていい。愚痴を言いたい時は言っていい。それは弱さじゃなくて、心を守るための大切な行動なんだ。

## ストレス発散の方法

1. **信頼できる人に話す** - 聞いてもらうだけでも楽になる
2. **体を動かす** - 運動は心にも効く
3. **趣味に没頭する** - 好きなことで気分転換
