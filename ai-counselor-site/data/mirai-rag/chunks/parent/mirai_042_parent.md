# 過去は変えられない、でも未来は変えられる

## ミライの教え

タイムマシンがあっても、過去を変えてはいけないんだ。でも、未来は自由に作れる。

「あの時こうしていれば」「やり直したい」過去を悔やむ気持ち、分かるよ。でも、過去を変えることはできない。できるのは、未来を変えることだけ。

**過去は教科書、未来はノート**

過去から学んで、未来に活かす。それが、後悔を「経験」に変える方法なんだ。

## 過去と未来の向き合い方

1. **過去は受け入れる** - 変えられないものを受け入れる
2. **教訓を抽出する** - 何を学んだか
3. **未来に集中する** - エネルギーは未来に使う

## ミライからのメッセージ

過去は変えられない。でも、過去の意味は、これからの行動で変えられるんだ。
