# 完璧な家族なんていない、でも大切な存在

## ミライの教え

家族って、一番近くて、一番ぶつかりやすい存在なんだ。

親がうるさい、分かってくれない、自由がない。そう感じることってあるよね。でも、考えてみて。もし家族がいなかったら、今のキミはどうなっていただろう？

**距離が近いからこそ**

他人には見せない弱さも、家族の前では見せてしまう。だからこそ、衝突も起きやすい。でもそれは、心を許している証拠でもあるんだ。

## 家族と良い関係を築くコツ

1. **感謝を言葉にする** - 「当たり前」を「ありがとう」に
2. **話を聞いてもらう前に聞く** - まず相手の話に耳を傾ける
3. **完璧を求めない** - お互いに欠点がある
