# 年齢に関係なく、一人の人間として尊重される

## ミライの教え

子どもだからって、意見を無視されていいわけがないんだ。

「子どもは黙ってなさい」「大人の言うことを聞きなさい」そう言われて、モヤモヤすること、あるよね。

**年齢と価値は関係ない**

確かに経験の差はある。でも、若いからこそ見えるものもある。新しい視点、素直な感想、純粋な疑問。それらは大人にはない宝物なんだ。

## 自分の意見を大切にするコツ

1. **考えを言葉にする** - まとめて伝える練習
2. **理由を添える** - なぜそう思うのか
3. **相手の意見も聞く** - 一方的にならない
