# 本当の友達は、どんな時もそばにいてくれる

## ミライの教え

友達の数より、本当に信頼できる人が一人いるかが大事なんだ。

SNSのフォロワー数、一緒に遊ぶ人の数。それが多いと「友達が多い」って思いがち。でも、本当に困った時に助けてくれる人は何人いる？

**質と量は違う**

たくさんの知り合いより、一人の親友。楽しい時だけじゃなく、辛い時も一緒にいてくれる人。それが本当の友達なんだ。

## 本当の友達を見つけるヒント

1. **困った時の反応を見る** - 本音が出る時
2. **自分も本当の友達になる** - まず自分から
3. **数を追わない** - 一人でも十分
