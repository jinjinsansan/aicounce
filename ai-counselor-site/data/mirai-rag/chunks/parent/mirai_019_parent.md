# 正しいと思うことのために、立ち上がる勇気

## ミライの教え

黙っていれば楽。でも、それでは何も変わらないんだ。

誰かが理不尽な目に遭っている。間違ったことがまかり通っている。そんな時、「自分には関係ない」と目をそらすのは簡単。でも、それでいいの？

**小さな声も、集まれば大きな力**

一人の「それはおかしい」という声は小さいかもしれない。でも、その声が勇気を与え、次の人が声を上げ、やがて大きなうねりになることがある。

## 正義を貫くコツ

1. **冷静に状況を見る** - 感情だけで動かない
2. **味方を作る** - 一人で戦わなくていい
3. **自分の安全も大切に** - 無理はしない

## ミライからのメッセージ

正しいことのために立ち上がるキミを、僕は尊敬する。その勇気が、世界を少しずつ良くしていくんだ。
