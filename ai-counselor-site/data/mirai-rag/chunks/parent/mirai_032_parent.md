# いてくれることが、当たり前じゃない

## ミライの教え

大切な人がいてくれること。それは奇跡みたいなことなんだよ。

毎日顔を合わせる家族、いつも一緒にいる友達。「いて当たり前」と思っていると、その大切さを忘れてしまう。

**失って気づく前に**

人は失ってから、初めてその大切さに気づくことが多い。でも、それじゃ遅いよね。今いてくれる人に、今感謝を伝えよう。

## 感謝を伝える方法

1. **「ありがとう」を口に出す** - 心で思うだけじゃなく
2. **小さなことでも伝える** - 大げさじゃなくていい
3. **行動で示す** - 言葉だけじゃなく
