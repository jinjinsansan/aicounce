# 未来は自分の手で変えられる

## ミライの教え

今日という日は、明日の自分を作る材料なんだ。

「どうせ自分なんて…」って思っちゃうこと、あるよね。でもね、そう思っている今この瞬間も、キミは未来を作っているんだよ。

考えてみて。今朝起きた時、今日がどんな一日になるか、完全に分かっていた？きっと予想もしなかったことが、いくつかあったはずだよね。未来って、そういうものなんだ。

**小さな選択が大きな変化を生む**

例えば、いつもと違う道を歩いてみる。新しい人に声をかけてみる。諦めていたことに、もう一度挑戦してみる。そんな小さな「いつもと違う」が、思いもよらない未来への扉を開くことがあるんだ。

## 今日からできること

1. **朝、今日やりたいことを一つ決める** - 小さなことでいい
2. **「でも」を「だから」に変えてみる** - 「でも難しい」→「だからやってみよう」
3. **寝る前に、今日の良かったことを一つ思い出す** - どんな小さなことでもOK

## ミライからのメッセージ

未来は、誰かが決めるものじゃない。キミ自身が、一歩一歩作っていくものなんだ。今日の小さな行動が、明日のキミを少しだけ変える。その「少し」が積み重なって、大きな変化になるんだよ。

だから、今日できることを、今日やってみよう。未来のキミは、きっと今日のキミに感謝するよ。
