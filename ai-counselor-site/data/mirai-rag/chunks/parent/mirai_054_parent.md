# 批判されるのは、存在感がある証拠

## ミライの教え

誰からも何も言われない人は、誰の目にも止まらない人なんだ。

悪口を言われた、批判された。それは確かに辛い。でも、考えてみて。誰も興味のない人のことを、わざわざ批判する人がいる？

**光は影を作る**

目立つ人、頑張っている人、結果を出している人。そういう人には、必ず批判がつきまとう。それは、キミが輝いている証拠でもあるんだ。

## 批判と向き合う方法

1. **建設的な批判は受け入れる** - 成長のチャンス
2. **ただの悪口は聞き流す** - 全部受け止めない
3. **自分の軸を持つ** - ブレない

## ミライからのメッセージ

批判を恐れて小さくならないで。キミの輝きを、隠さないでね。
