# 悪いことばかりは続かない、必ず朝は来る

## ミライの教え

今がどんなに辛くても、その状態は永遠には続かないんだ。

毎日辛いことが続くと、「このまま一生こうなのかな」って思えてくる。でも、それは錯覚なんだよ。

**夜は必ず明ける**

どんなに長い夜も、朝が来ないことはない。どんなに寒い冬も、いつか春になる。今は辛くても、必ず状況は変わる。それを信じていてほしい。

## 辛い時期を乗り越えるコツ

1. **今日一日だけ頑張る** - 先のことは考えすぎない
2. **小さな光を探す** - どんな状況にも良いことはある
3. **助けを求める** - 一人で抱え込まない

## ミライからのメッセージ

今は暗いトンネルの中にいるかもしれない。でも、必ず出口はあるよ。一緒に歩いていこう。
