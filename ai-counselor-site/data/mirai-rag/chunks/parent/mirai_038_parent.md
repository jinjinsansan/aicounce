# まだまだ頑張れる、もっとできるはず

## ミライの教え

「もう限界」と思った時、実はまだ余力があることが多いんだ。

「これ以上は無理」「もう頑張った」そう思う時ってあるよね。でも、本当にそう？

**限界は自分で決めている**

多くの「限界」は、体の限界じゃなくて心の限界。「もう無理」と思った先に、実はまだ力が残っていることが多いんだ。

## もう一歩踏み出すコツ

1. **「あと少しだけ」と唱える** - 小さく目標を設定
2. **休憩を挟む** - 完全に止めないで
3. **なぜ頑張りたいか思い出す** - 目的を再確認
