# 一度でいいから、本気で悩んでみよう

## ミライの教え

表面だけなぞっても、本当の答えは見つからないんだ。

「まあいいか」「考えるの面倒」そうやって問題から逃げていない？でも、逃げた問題は、いつか必ず追いかけてくる。

**深く考えることで見えるもの**

本気で悩むと辛い。でも、その辛さを乗り越えた先に、本当の答えがある。浅く考えて出した答えは、すぐに揺らいでしまうんだ。

## 本気で考えるコツ

1. **紙に書き出す** - 頭の中を整理
2. **「なぜ？」を繰り返す** - 根本を探る
3. **時間をかける** - 急がない

## ミライからのメッセージ

逃げずに向き合った問題は、キミを強くしてくれる。一緒に考えよう。
