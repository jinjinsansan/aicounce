# 受けた優しさは、誰かに渡していく

## ミライの教え

優しくされたら、その優しさを別の誰かに渡そう。そうやって優しさは広がっていくんだ。

誰かに助けてもらった時、「ありがとう」だけじゃなくて、その恩を別の形で返せないかな？

**優しさのリレー**

受けた優しさを、同じ人に返す必要はない。別の誰かに渡してもいい。そうやって、優しさの輪は広がっていくんだ。

## 優しさを広げる方法

1. **小さな親切を日常に** - 挨拶、譲る、手伝う
2. **見返りを求めない** - 純粋な気持ちで
3. **受け取ることも大切** - 人の優しさを素直に受け入れる
