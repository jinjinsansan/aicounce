# 悩んでいる暇があったら、一つでもやってみよう

## ミライの教え

完璧な準備ができるのを待っていたら、一生始められないよ。

「失敗したらどうしよう」「もう少し準備してから」「タイミングが…」頭の中でぐるぐる考えているうちに、時間だけが過ぎていく。そんな経験、ない？

**考えることと悩むことの違い**

考えることは建設的。「どうすればうまくいくか」を前向きに検討すること。悩むことは堂々巡り。同じことを何度も繰り返し考えて、結局何も進まないこと。

もし今、悩みのループにハマっていると感じたら、それは「行動のサイン」なんだ。

## 行動を起こすコツ

1. **5秒ルール** - 思い立ったら5秒以内に動き出す
2. **最初の一歩だけ決める** - 全部考えなくていい
3. **やりながら考える** - 走りながら方向を修正

不完全でいい。とりあえずやってみよう。やってみて初めて分かることが、たくさんあるんだから。
