# 恋をして、傷ついて、それでも人を好きになる

## ミライの教え

恋は嬉しいことばかりじゃない。でも、それでも人を好きになる価値はあるんだ。

好きな人ができて、でも振り向いてもらえなくて、胸が痛い。「もう恋なんてしたくない」そう思うこともあるよね。

**傷つくことを恐れない**

恋をしなければ傷つかない。でも、恋をしなければ得られない喜びもある。傷ついた分だけ、人は優しくなれるんだよ。

## 恋の痛みと向き合う方法

1. **感情を否定しない** - 悲しい時は悲しんでいい
2. **時間を味方に** - 必ず和らぐ
3. **経験として受け入れる** - 次の恋の糧になる

## ミライからのメッセージ

恋をしたキミは、素敵だよ。たとえ結果がどうであれ、その気持ちは本物なんだから。
