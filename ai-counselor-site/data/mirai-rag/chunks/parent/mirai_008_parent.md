# 過ぎた時間は戻ってこない

## ミライの教え

時間は誰にも平等に与えられた、最も貴重な資源なんだ。

お金は失っても取り戻せる。物は壊れても買い直せる。でも、時間だけは一度過ぎたら二度と戻ってこない。1秒1秒が、取り返しのつかない大切な瞬間なんだよ。

**「いつか」は永遠に来ない**

「いつかやろう」「時間ができたら」そう思っていることって、ある？でもね、その「いつか」は待っていても来ないんだ。時間は「できる」ものじゃなくて「作る」もの。

## 時間を大切にする方法

1. **「今日」でできることを「明日」に回さない**
2. **本当に大切なことに時間を使う**
3. **「忙しい」を言い訳にしない**
