# 楽しかった思い出は、永遠になくならない

## ミライの教え

時間は過ぎても、その時感じた気持ちは心の中に残るんだ。

「あの頃は良かった」「あの時に戻りたい」そう思うこと、あるよね。でも、戻れないからこそ、思い出は輝くんだ。

**思い出は力になる**

辛い時、苦しい時、楽しかった思い出が支えてくれることがある。「あんな風に笑えた自分がいる」その事実が、また笑える未来への希望になる。

## 思い出を大切にする方法

1. **写真や記録を残す** - 後で振り返れるように
2. **一緒に過ごした人に感謝** - 思い出は一人じゃ作れない
3. **今を楽しむ** - 今日が明日の思い出になる
