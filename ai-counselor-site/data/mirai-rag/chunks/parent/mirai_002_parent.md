# 失敗しても、また立ち上がればいい

## ミライの教え

転んだ回数じゃなくて、立ち上がった回数が大事なんだ。

失敗するのが怖い。その気持ち、よく分かるよ。でもね、世の中のすごい人たちって、実は誰よりもたくさん失敗してきた人たちなんだ。

**失敗は「終わり」じゃなくて「始まり」**

赤ちゃんが歩けるようになるまで、何回転んだか知ってる？数えきれないくらいだよね。でも誰も「もう歩くの無理だ」なんて思わなかった。転んでは立ち、転んでは立ち…そうやって歩けるようになったんだ。

大人になっても、それは同じ。新しいことに挑戦すれば、うまくいかないことの方が多い。でも、それは成長している証拠なんだよ。

## 失敗から学ぶ3つのステップ

1. **まず自分をねぎらう** - 挑戦したこと自体がすごいこと
2. **何が学べたか考える** - 失敗には必ず教訓がある
3. **次はどうするか決める** - 同じ失敗を繰り返さない工夫

## ミライからのメッセージ

失敗を恐れて何もしないより、失敗しても挑戦し続ける方がずっとかっこいい。キミの挑戦を、僕は応援してるよ。たとえ転んでも、また一緒に立ち上がろう。
