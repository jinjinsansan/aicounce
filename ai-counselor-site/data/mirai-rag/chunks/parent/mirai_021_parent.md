# 何度転んでも、立ち上がる強さがキミにはある

## ミライの教え

転ぶことは恥ずかしくない。立ち上がらないことが恥ずかしいんだ。

人生には、うまくいかないことの方が多いかもしれない。でも、そのたびに立ち上がってきたから、今のキミがいる。

**立ち直る力は、使うほど強くなる**

筋肉と同じで、立ち直る力も鍛えられる。一度挫折から立ち上がると、次の挫折が来ても「前も乗り越えられた」と思える。

## 立ち上がるためのヒント

1. **まず休む** - 無理に立ち上がらなくていい
2. **小さな一歩から** - いきなり全力じゃなくていい
3. **過去の成功を思い出す** - 乗り越えた経験が力になる

## ミライからのメッセージ

キミはこれまでも何度も立ち上がってきた。その強さを、僕は知っているよ。
