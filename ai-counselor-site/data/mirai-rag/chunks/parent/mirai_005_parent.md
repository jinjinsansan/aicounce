# 毎日の小さな積み重ねが、大きな力になる

## ミライの教え

山を動かすのは、一度の大きな力じゃない。毎日コツコツ運ぶ小さな石なんだ。

「もっと頑張らなきゃ」「一気に変わりたい」そう思って、無理して疲れちゃったこと、ない？でもね、本当の変化って、そんな劇的なものじゃないんだよ。

**1%の成長を365日続けると？**

毎日たった1%だけ成長する。それを1年続けると、最初の37倍以上になる計算になるんだ。逆に、毎日1%サボると、1年後には最初の3%くらいまで落ちてしまう。

小さな差が、時間をかけて大きな差になる。これが「継続の力」なんだ。

## 継続するためのコツ

1. **ハードルを下げる** - 「毎日1時間勉強」より「毎日1ページ」
2. **習慣にくっつける** - 「歯磨きの後に」など既存の習慣と組み合わせ
3. **完璧を求めない** - できない日があっても自分を責めない

## ミライからのメッセージ

今日できる小さな一歩を、明日も、明後日も続けてみよう。気づいた時には、想像もしなかった場所にたどり着いているはずだよ。
