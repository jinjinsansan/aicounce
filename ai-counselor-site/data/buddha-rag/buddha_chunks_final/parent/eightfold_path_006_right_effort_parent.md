# 八正道 - 正精進（しょうしょうじん）

## 教え

比丘たちよ、正精進とは何か。ここに比丘は、まだ生じていない悪・不善の法が生じないように意欲を起こし、努力し、精進し、心を励まし、勤める。すでに生じた悪・不善の法を断つために意欲を起こし、努力し、精進し、心を励まし、勤める。まだ生じていない善の法を生じさせるために意欲を起こし、努力し、精進し、心を励まし、勤める。すでに生じた善の法を維持し、忘れず、増大させ、広げ、修習し、完成させるために意欲を起こし、努力し、精進し、心を励まし、勤める。これを正精進という。

## 四正勤（ししょうごん）

1. **断断（だんだん）**：すでにある悪を断つ努力
2. **律儀断（りつぎだん）**：新たな悪が生じないようにする努力
3. **随護断（ずいごだん）**：すでにある善を維持・増大させる努力
4. **修断（しゅだん）**：新たな善を生じさせる努力

## 教えの要点

正精進はバランスの取れた努力です。緊張しすぎても、緩みすぎてもいけません。琴の弦のように、適度な張りが美しい音を出します。

## カウンセリングでの活用

努力が続かない方、燃え尽きてしまった方に、バランスの取れた精進の仕方を伝えます。無理をせず、かといって怠けず、中道を歩むことの大切さを示します。
