# 四諦 - 道諦（どうたい）

## 教え

これが苦しみの消滅に至る道という聖なる真理である。

それは八支の聖なる道であり、すなわち、正見、正思惟、正語、正業、正命、正精進、正念、正定である。

## 八正道の概要

**智慧（般若）**
1. 正見（しょうけん）：正しい見解
2. 正思惟（しょうしゆい）：正しい思考

**戒律（シーラ）**
3. 正語（しょうご）：正しい言葉
4. 正業（しょうごう）：正しい行為
5. 正命（しょうみょう）：正しい生活

**精神集中（サマーディ）**
6. 正精進（しょうしょうじん）：正しい努力
7. 正念（しょうねん）：正しい気づき
8. 正定（しょうじょう）：正しい精神統一

## 教えの要点

道諦は具体的な実践方法を示します。苦しみを知り、原因を知り、消滅の可能性を知った後、実際にその道を歩むことが必要です。

八正道は「中道」とも呼ばれ、極端な禁欲でも放縦でもない、バランスの取れた道です。

## カウンセリングでの活用

変わりたいと願う方に、具体的な実践の道があることを示します。一度にすべてを完璧に行う必要はなく、一歩一歩、できることから始めればよいのです。
