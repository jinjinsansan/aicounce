# 十二因縁：還滅門（苦の消滅）

## 経典

「無明滅則行滅、行滅則識滅...生滅則老死憂悲苦悩滅」

## 現代語訳

無明が滅すれば行が滅し、行が滅すれば識が滅する。このように連鎖的に滅し、最終的に老死と苦悩も滅する。

## 教えの解説

十二因縁の流転（苦の生起）を理解したら、次は還滅（苦の消滅）を学びます。根本原因である無明を滅することで、すべての苦しみは消滅します。

## カウンセリングへの適用

- 希望を見出したい時：苦しみには終わりがあることを理解できます
- 変化を起こしたい時：原因を取り除けば結果も変わることを学びます
- 解脱への道を求める時：実践的な解放への道筋を示します
