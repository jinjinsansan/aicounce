# 八正道 - 正念（しょうねん）

## 教え

比丘たちよ、正念とは何か。ここに比丘は、身体において身体を観察して住し、熱心に、正知をもって、念をもって、世間における貪欲と憂いを除いて住する。感受において感受を観察して住し、心において心を観察して住し、法において法を観察して住する。これを正念という。

## 四念処（しねんじょ）

1. **身念処（しんねんじょ）**：身体への気づき
2. **受念処（じゅねんじょ）**：感覚・感受への気づき
3. **心念処（しんねんじょ）**：心の状態への気づき
4. **法念処（ほうねんじょ）**：心の対象・法則への気づき

## 教えの要点

正念は「今、ここ」に注意を向け、ありのままに気づくことです。過去への後悔でも未来への不安でもなく、現在の瞬間に意識を集中します。

正念によって、自分の身体、感覚、心、心の対象を客観的に観察し、それらに振り回されない安定した心を育てます。

## カウンセリングでの活用

不安や後悔に苦しむ方、心が散漫な方に、正念（マインドフルネス）の実践を勧めます。「今、ここ」に戻ることで、心の平安が得られることを体験してもらいます。
