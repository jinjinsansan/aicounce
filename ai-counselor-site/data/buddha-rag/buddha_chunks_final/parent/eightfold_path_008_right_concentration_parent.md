# 八正道 - 正定（しょうじょう）

## 教え

比丘たちよ、正定とは何か。ここに比丘は、諸欲を離れ、不善の法を離れて、尋と伺を伴い、離から生じた喜と楽のある初禅に入って住する。尋と伺の止息により、内に清浄あり、心が一点に集中し、尋と伺を離れ、定から生じた喜と楽のある第二禅に入って住する。喜の消退により、捨をもって住し、正念正知あり、身体で楽を経験し、聖者たちが「捨あり、念あり、楽住」と説く第三禅に入って住する。楽の断と苦の断により、すでに先に喜と憂が消滅しているので、不苦不楽であり、捨による念の清浄ある第四禅に入って住する。これを正定という。

## 四禅（しぜん）

1. **初禅**：欲と不善から離れ、喜と楽がある状態
2. **第二禅**：心が一点に集中し、内なる清浄がある状態
3. **第三禅**：喜が消え、楽と捨がある状態
4. **第四禅**：苦楽を超えた、完全な平静の状態

## 教えの要点

正定は深い瞑想の状態です。心が一つの対象に集中し、散乱がなくなり、深い平安と智慧が生まれます。

正定は他の七つの道が整った上で達成されます。戒律と智慧という土台がなければ、真の精神統一は得られません。

## カウンセリングでの活用

心の散乱に苦しむ方、深い平安を求める方に、瞑想の実践を勧めます。段階的に心を落ち着ける方法を伝え、焦らず継続することの大切さを示します。
