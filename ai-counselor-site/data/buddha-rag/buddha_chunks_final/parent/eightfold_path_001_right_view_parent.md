# 八正道 - 正見（しょうけん）

## 教え

比丘たちよ、正見とは何か。苦を知ること、苦の生起を知ること、苦の消滅を知ること、苦の消滅に至る道を知ること。これを正見という。

## 解説

正見は八正道の出発点であり、智慧の根本です。

**正見の内容**
- 四諦（苦・集・滅・道）を理解すること
- 因果の法則（業）を理解すること
- 無常・苦・無我の三相を理解すること
- 善悪の区別を正しく知ること

## 教えの要点

正見とは、物事をありのままに見る智慧です。私たちは往々にして、欲望や怒り、無知によって曇った目で世界を見ています。

正見を持つことで、苦しみの本質、その原因、解決の可能性、そして具体的な道を明確に理解できます。

## カウンセリングでの活用

認知の歪み（白黒思考、過度の一般化、自己否定など）に苦しむ方に、物事を偏りなく見る視点を提供します。「正しく見る」ことが、心の変容の第一歩です。
