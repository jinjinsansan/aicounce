# 四念処：気づきの修行

## 経典

「比丘らよ、この道は衆生の清浄と、悲嘆と嘆きの超越と、苦と憂いの消滅と、正理の獲得と、涅槃の実証のための唯一の道である。すなわち四念処である」

## 現代語訳

この修行道は、生きとし生けるものが清らかになり、悲しみや嘆きを乗り越え、苦しみと憂いを滅し、悟りを得て涅槃を実現するための唯一の道です。

## 四念処の内容

1. **身念処**：身体への気づき
2. **受念処**：感受への気づき
3. **心念処**：心への気づき
4. **法念処**：法（現象）への気づき

## カウンセリングへの適用

- 不安で落ち着かない時：今この瞬間に意識を向ける方法を学びます
- 雑念に振り回される時：思考を観察することで距離を置けます
- 自分を見失っている時：身体と心への気づきで自己を取り戻します
