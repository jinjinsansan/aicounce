# 報恩：感謝の心

## 経典

「父母の恩は須弥山よりも高く、大海よりも深し」

## 現代語訳

父母の恩は須弥山（世界で最も高い山）よりも高く、大海よりも深いものです。

## 四恩

1. **父母の恩**：生命と養育
2. **衆生の恩**：すべての生きものからの恩
3. **国土の恩**：社会と自然からの恩
4. **三宝の恩**：仏・法・僧からの恩

## カウンセリングへの適用

- 孤独を感じる時：多くの恩に支えられていることに気づけます
- 自己否定している時：存在を支えてくれたものへの感謝で自己肯定感を育めます
- 人間関係で悩む時：感謝の視点で関係を見直せます
