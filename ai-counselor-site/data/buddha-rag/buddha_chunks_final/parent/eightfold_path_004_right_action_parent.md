# 八正道 - 正業（しょうごう）

## 教え

比丘たちよ、正業とは何か。殺生を離れること、偸盗を離れること、邪淫を離れること。これを正業という。

## 三つの正業

1. **不殺生（ふせっしょう）**：生き物を殺さない、命を尊重する
2. **不偸盗（ふちゅうとう）**：盗まない、与えられていないものを取らない
3. **不邪淫（ふじゃいん）**：不倫や性的な不正を行わない

## 教えの要点

正業は身体の行為に関する教えです。行為は心の表れであり、同時に心を形成します。善い行いは善い心を育て、悪い行いは心を汚します。

正業の根本は「害を与えない」ことと「他者の権利を尊重する」ことです。これは消極的な禁止ではなく、積極的に命を守り、分かち合い、誠実であることを意味します。

## カウンセリングでの活用

行動面での問題（暴力、窃盗、不倫など）を抱える方に、行為と心の関係を理解してもらいます。行動を変えることで心も変わり、心が変われば行動も自然と変わることを伝えます。
