# 阿弥陀経 - 極楽浄土の荘厳

## 経文要旨

これより西方、十万億の仏土を過ぎて、世界あり、名づけて極楽という。その土に仏まします、阿弥陀と号す。

彼の国には七重の欄楯、七重の羅網、七重の行樹あり。皆、四宝をもって周匝囲繞せり。また、七宝の池あり。八功徳水その中に充満せり。

## 極楽浄土の特徴

- 苦しみがない
- 清らかで美しい
- 仏法を学べる
- 悟りに向かって進める
- すべての望みが叶う

## 教えの要点

極楽浄土は、阿弥陀仏が建立した理想の世界です。そこには苦しみがなく、すべての条件が整い、悟りへの道を歩むことができます。

現世で完全な悟りを得られなくても、極楽浄土に往生すれば、必ず悟りに至ることができます。

## カウンセリングでの活用

死や来世への不安を持つ方に、極楽浄土という希望を伝えます。念仏を通して心の安らぎを得る方法を紹介します。
