# 捨離：手放しの智慧

## 経典

「執着を離れ、取ることなく、捨てることなく、ありのままに見る者は、苦しみより解放される」

## 現代語訳

執着から離れ、何かを取ろうとせず、また無理に捨てようともせず、物事をありのままに見ることができる者は、苦しみから解放されます。

## 教えの解説

手放しは無関心ではありません。物事を深く理解した上で、執着せずに関わることです。握りしめることをやめれば、手は自由になります。

## カウンセリングへの適用

- 過去にとらわれている時：手放すことで前に進める力を得られます
- 失ったものを惜しむ時：執着と愛情の違いを理解できます
- コントロールを手放せない時：委ねることの平安を学べます
