# 八正道 - 正語（しょうご）

## 教え

比丘たちよ、正語とは何か。嘘を離れること、離間語を離れること、粗悪語を離れること、綺語を離れること。これを正語という。

## 四つの正語

1. **不妄語（ふもうご）**：嘘をつかない、真実を語る
2. **不両舌（ふりょうぜつ）**：二枚舌を使わない、人を離間させない
3. **不悪口（ふあっく）**：荒い言葉を使わない、優しく語る
4. **不綺語（ふきご）**：無駄話、意味のない言葉を避ける

## 教えの要点

言葉は剣よりも鋭く、人を傷つけることも癒すこともできます。正語は単に「悪い言葉を避ける」だけでなく、真実で、和合をもたらし、優しく、意味のある言葉を使うことです。

言葉は私たちの心を反映すると同時に、心を形成します。正しい言葉を使うことで、心もまた正されていきます。

## カウンセリングでの活用

言葉による問題（口論、悪口、嘘など）を抱える方に、言葉の力と影響を理解してもらいます。言葉を変えることで、人間関係も心も変わっていくことを伝えます。
