# 八正道 - 正命（しょうみょう）

## 教え

比丘たちよ、正命とは何か。聖なる弟子は邪な生活を捨てて、正しい方法で生計を立てる。これを正命という。

## 避けるべき生業

1. 武器の売買
2. 生き物の売買（奴隷、家畜など）
3. 肉の売買
4. 酒・毒物の売買
5. 毒物の売買

## 教えの要点

正命は、仕事や生計の立て方に関する教えです。一日の大半を費やす仕事が、他者を害したり、不正に基づいたりしていれば、心の平安は得られません。

現代においては、自分の仕事が社会や環境、他者に害を与えていないか、正直で公正な方法で生計を立てているかを問うことです。

## カウンセリングでの活用

仕事の倫理的な問題や、生き方の方向性に悩む方に、正命の視点を提供します。仕事は単なる収入手段ではなく、修行の一部であり、心の平安と深く関わっていることを伝えます。
