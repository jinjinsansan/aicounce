# 正語の日常実践

## 話す前の四つの問い
1. これは真実か？
2. これは親切か？
3. これは必要か？
4. これは適切な時か？

## 実践のポイント
- 怒りの中では話さない
- 相手の立場で言葉を選ぶ
- 沈黙も正語の一つ
