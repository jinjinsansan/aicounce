# 正見の内容

## 正見が含むもの
1. **因果の理解**：行為には結果が伴う
2. **四諦の理解**：苦・集・滅・道
3. **三法印の理解**：無常・苦・無我
4. **善悪の区別**：何が善で何が悪か

## 実践
物事をありのままに見る。先入観や偏見を離れて、真実を見極める目を養う。
