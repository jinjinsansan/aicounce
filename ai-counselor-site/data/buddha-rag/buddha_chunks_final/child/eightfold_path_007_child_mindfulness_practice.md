# 正念の日常実践

## 四念処の実践
1. **身体への気づき**：呼吸、歩行、動作
2. **感受への気づき**：快・不快・中性
3. **心への気づき**：心の状態を観察
4. **法への気づき**：現象の生滅を観察

## 簡単な実践
- 呼吸に3回意識を向ける
- 「今、〜している」と心で確認
- 感情に名前をつける
