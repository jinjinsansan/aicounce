# 怒りへの仏教的アプローチ

## 怒りの理解
怒りは「こうあるべき」が裏切られた時に生じる。

## RAIN法
- **R (Recognize)**：怒りを認識する
- **A (Accept)**：怒りがあることを認める
- **I (Investigate)**：何が原因か調べる
- **N (Non-identify)**：「私は怒り」ではなく「怒りがある」

## 即効性のある対処
1. 口を閉じる（言葉にしない）
2. 10数える
3. その場を離れる
4. 深呼吸する
