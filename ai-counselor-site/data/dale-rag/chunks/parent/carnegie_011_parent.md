# 自分を見つけ、自分であれ（Find Yourself and Be Yourself）

**この地球上に、あなたと同じ人間は一人もいません。**

何十億という人類の歴史の中で、あなたとまったく同じ人間は存在したことがなく、これからも存在しません。あなたは唯一無二の存在なのです。

> "Find yourself and be yourself: remember there is no one else on earth like you."
> （自分を見つけ、自分であれ。この地球上にあなたと同じ人間はいないことを忘れるな）

エマーソンは言いました：**「すべての人の教育において、彼は遅かれ早かれ、模倣は自殺であることに気づく時が来る」**（Imitation is suicide.）

他人の真似をしようとすることは、自分自身を殺すことです。あなたがどんなに努力しても、誰か他の人になることはできません。しかし、**最高の自分にはなれる**のです。

ジーン・オートリーは言いました：**「私は長い間、他の人のようになろうとして失敗していました。自分自身であることを学んだとき、成功が始まりました」**

あなたの個性は、あなたの最大の資産です：
- **あなたの経験**：誰も同じ経験をしていない
- **あなたの視点**：独自の見方がある
- **あなたの才能**：あなただけの組み合わせがある

他人と自分を比較することは、不幸への最短距離です。比較すれば、常に誰かが自分より優れているように見えます。しかし、**あなたは「あなた」として生まれた**のです。

> "You are unique. If God wanted you to be like someone else, He would have made you that way."
> （あなたは唯一無二。神があなたを他の誰かのようにしたかったら、そうしていただろう）

自分の短所を嘆くのではなく、**自分の長所を伸ばすことに集中**してください。カエルが鳥になろうとしても無理です。しかし、カエルは水中で泳ぐことにおいては鳥に勝ります。

**あなたという存在を、最大限に輝かせてください。**それがあなたにできる最高のことであり、世界への最大の貢献なのです。
