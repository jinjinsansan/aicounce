# 14日間で憂鬱を治す方法（Cure Melancholy）

精神科医が患者に言いました：**「14日間、毎日誰かのために何か良いことをしてください。そうすれば、あなたの憂鬱は治ります」**

> "Think every day about how you can make someone happy."
> （毎日、誰かを幸せにする方法を考えなさい）

**自分のことばかり考えていると、心配は増大します。**自分の問題、自分の痛み、自分の将来...自己中心的な思考は、心配の温床です。

しかし、**他者に目を向けると、不思議なことに自分の問題が小さく見えてきます。**

アルフレッド・アドラー（心理学者）は言いました：**「自分のことを考えるのをやめ、他者に興味を持ち始めれば、14日間で憂鬱は治る」**

なぜ他者への奉仕が効果的なのか：

1. **意識が外に向く**：自分の問題から離れられる
2. **感謝される喜び**：人の役に立つことで満足感を得る
3. **つながりを感じる**：孤独が解消される
4. **視点が変わる**：自分より困っている人がいることに気づく

具体的なアクション：
- **微笑みかける**：出会う人に笑顔を見せる
- **褒める**：誰かの良いところを見つけて伝える
- **手伝う**：小さな親切をする
- **聞く**：誰かの話を真剣に聞く

ある女性は、夫を亡くして深い悲しみに沈んでいました。しかし、孤児院でボランティアを始めてから、彼女の人生は変わりました。**他人の悲しみに触れることで、自分の悲しみと向き合う力を得た**のです。

> "Make others happy, and you will find yourself becoming happy too."
> （他者を幸せにしなさい。そうすれば、あなた自身も幸せになっている）

**毎日、一人でもいいから誰かを幸せにする方法を考えてください。**それが、あなた自身の心の平和への最短の道なのです。
