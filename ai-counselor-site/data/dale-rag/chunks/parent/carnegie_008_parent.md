# 心配に「損切りライン」を設定する（Stop-Loss Order）

株式市場では、投資家は**「損切り注文」（Stop-Loss Order）**を使います。株価が一定の価格まで下がったら、自動的に売却して損失を限定するのです。

**私たちの心配事にも「損切りライン」を設定すべきです。**

> "How much anxiety is this thing worth? At what point shall I set a stop-loss order and forget about it?"
> （この件はどれだけの心配に値するか？どの時点で損切りして忘れるべきか？）

ある男性は、隣人との些細なトラブルに何年もエネルギーを費やしていました。裁判を起こし、何時間も怒りに費やし、健康まで害しました。最終的に勝訴しましたが、得たのはわずか数ドル。**失ったものは計り知れない**—時間、健康、心の平和。

自問してください：
- **この問題は、私が費やしている心配に本当に値するか？**
- **どの時点で「もう十分だ」と言って前に進むべきか？**
- **この問題に費やすエネルギーを、もっと生産的なことに使えないか？**

人生には「損失」を認めて先に進むべき時があります：
- 関係がうまくいかないとき
- ビジネスが失敗したとき
- 過去の決断を後悔しているとき

**沈みゆく船にしがみついていては、一緒に沈むだけです。**

賢い投資家は、感情に流されず、あらかじめ決めた損切りラインで売却します。同じように、**心配事にもあらかじめ限度を設定し、それを超えたら手放す**のです。

時には「負け」を認めることが、実は最大の「勝ち」なのです。執着を手放し、新しい可能性にエネルギーを向けましょう。
