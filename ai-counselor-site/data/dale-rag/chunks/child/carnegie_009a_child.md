# おがくずを挽こうとするな

過去は変えられない。「あの時こうしていれば」と悔やむ時間は、現在と未来を改善する時間を奪う。過去から学んだら前に進む。「死んだ過去は死者に葬らせよ」。今日が残りの人生の最初の日。かけがえのない時間を過去のおがくずを挽くことに浪費しない。
