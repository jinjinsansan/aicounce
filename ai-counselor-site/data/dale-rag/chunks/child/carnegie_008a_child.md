# 心配に損切りラインを設定する

株の損切り注文のように、心配事にも限度を決める。自問する：「この問題は費やしている心配に本当に値するか？」「どの時点で前に進むべきか？」沈みゆく船にしがみついては一緒に沈む。時には「負け」を認めることが最大の「勝ち」。執着を手放し、新しい可能性へ。
