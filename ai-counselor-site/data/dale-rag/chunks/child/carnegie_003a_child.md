# 心配が健康を蝕む

「心配と闘う術を知らないビジネスマンは若くして死ぬ」（カレル博士）。医者を訪れる患者の70%は恐れや心配を取り除くだけで治せる。胃潰瘍は何を食べるかではなく、何があなたを食い尽くしているかで起こる。心配は心臓病、高血圧、胃潰瘍、不眠の原因となる。
