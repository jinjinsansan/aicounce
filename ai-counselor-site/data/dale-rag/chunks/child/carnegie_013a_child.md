# 死んだ犬を蹴る人はいない

批判されるのは批判する価値があるから。リンカーンも「猿」「馬鹿」と呼ばれた。不当な批判は褒め言葉の変装。正当な批判なら改善に活かす。最善を尽くしたら批判という雨から傘を差す。すべての人を満足させることは不可能。批判は避けられないが、心の平和を奪う必要はない。
