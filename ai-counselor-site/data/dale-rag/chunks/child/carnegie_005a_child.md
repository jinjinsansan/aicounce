# 忙しくして心配を追い出す

計画と思考を必要とする何かに忙しくしていると、心配は難しい。チャーチルも「忙しすぎて心配する暇がない」と言った。心配という「悪い釘」を行動という「良い釘」で追い出す。活動に没頭すれば、ぐるぐると思い詰めることがなくなる。
