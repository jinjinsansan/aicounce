# 避けられないことに協力する

変えられないことと闘うのは無駄。柳は嵐に折れない—風と共にしなうから。エピクテトス：「幸福への道は一つ。自分の力の及ばないことについて悩むのをやめること」。受け入れることは諦めではない。変えられないことを受け入れ、変えられることにエネルギーを集中すること。
