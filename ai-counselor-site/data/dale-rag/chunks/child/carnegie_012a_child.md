# レモンをレモネードにしよう

人生がレモンを与えたら、レモネードを作れ。ベートーヴェンは聴力を失って最高の交響曲を書いた。「人生で最も重要なのは利益を活かすことではない。損失を活かすことだ。愚か者は良い手から勝つ。賢者は悪い手からも勝つ」。困難はあなたを破壊も強くもできる。選ぶのはあなた。
