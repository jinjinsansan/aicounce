# 14日間で憂鬱を治す方法

精神科医の処方：毎日誰かのために何か良いことをする。自分のことばかり考えると心配は増大するが、他者に目を向けると自分の問題が小さく見える。アドラー：「他者に興味を持てば14日で憂鬱は治る」。毎日一人でも誰かを幸せにする方法を考える—それが心の平和への最短の道。
