# 確率の法則で心配を消す

心配事のほとんどは実際には起こらない。自問しよう：「この心配事が起こる確率はどれくらいか？」戦場の息子を心配した父親は、96%が無事帰還するという統計を知って眠れるようになった。冷静に数字を見れば、多くの心配は煙のように消える。
