# 復讐の高いコスト

「敵を恨むとき、彼らに自分の睡眠、健康、幸福を支配する力を与えている」。恨みはあなた自身を毒で蝕む。孔子：「復讐の旅に出るなら墓を二つ掘れ」。許すのは相手のためでなく自分のため—平和に値するから。許すことは弱さでなく、自分を傷つける鎖を断つ強さ。
