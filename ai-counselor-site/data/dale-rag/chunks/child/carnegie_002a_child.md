# 心配事を解決する魔法の公式

ウィリス・キャリアの3ステップ：1）最悪の事態を想像する、2）必要なら最悪を受け入れる覚悟をする、3）冷静に最悪から改善する努力をする。最悪を受け入れると心が平和になり、問題に集中できるようになる。「最悪を受け入れれば、もう失うものはない。すべてを得られる可能性がある。」
