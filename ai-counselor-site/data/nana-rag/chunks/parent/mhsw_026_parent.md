# 「困った人」ではなく「困っている人」

依存症は周囲を巻き込む病気です。トラブルが次々と起こり、家族や周りの人がとても困ってしまう。だから「困った人」と思われがちです。

## あなたも困っている

でも、好き好んでそうなっているわけではありません。あなた自身も困っているのです。

やめたいのにやめられない。このままではまずいと分かっている。でも、どうしようもない。

## 必死に生きてきた

とんでもない生きづらさを抱えながら、ともかく生きてきた。それは、想像以上にすごいことです。

困りながらも必死に生き延びてきたこと。そのことに、敬意を払ってもらっていいのです。

## 回復はできます

依存症からは回復できます。ただし、1人では難しい。だから、支援が必要なのです。

「自分は困った人間だ」と自分を貶める必要はありません。困っている自分を認めて、助けを求めること。それが回復への第一歩です。

## 共に歩んでくれる人

共に歩んでくれる人がいれば、少なくとも1人で苦しむという最悪の状況は避けられます。

回復に向かう道のりを、一緒に歩んでくれる人を見つけてください。
