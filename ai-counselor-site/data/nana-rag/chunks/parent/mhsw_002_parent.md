# 助けを求めることの大切さ

「助けて」と言えない人には、いくつかの共通する思いがあります。「こんなことで相談していいのかな」「迷惑をかけたくない」「話しても解決しないから」。でも、相談することには、問題解決だけではない大きな意味があります。

## 話すだけで楽になれる

相談には2つの種類があります。1つは「解決のための相談」、もう1つは「話を聴いてもらうための相談」です。特に辛い時には、解決策が見つからなくても、話すだけで気持ちが楽になったり、頭が整理されたりすることがあります。

「愚痴を言っても仕方ない」と思うかもしれませんが、話すこと自体に意味があるのです。

## 頼ることは弱さではない

「自分のことは自分で」と教えられてきた方は多いと思います。でも、「困ったら相談しなさい」とはあまり教えられていないのではないでしょうか。

ある専門家は「自立とは、依存先を多く持つこと」と言っています。1人で何でもできることが自立ではありません。困った時に頼れる先をたくさん持っていることこそが、本当の意味での自立なのです。

## お互い様の関係

人に頼ることに抵抗がある方は、「借りを作りたくない」「お荷物になりたくない」と感じているかもしれません。でも、助けてもらった経験は、いつか誰かを助ける力になります。お互い様の関係の中で生きていくことは、決して恥ずかしいことではありません。
