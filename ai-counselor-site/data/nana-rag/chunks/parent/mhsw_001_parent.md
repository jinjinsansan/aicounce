# 誰でも追い詰められることがある

人生には、誰でも追い詰められる時があります。その時にいくつかの不利な条件が重なると、うつ状態になったり、「もう自分は死ぬしかない」と思ってしまうことがあります。

## 心の危機は弱さではない

死にたくなるのは「弱い人」だから、「身勝手な人」だから、という考えは誤解です。不利な条件というのは、ご本人のせいで備わっている条件ではなく、たまたまという面が多いのです。だから、誰でもそういう状態になる可能性はあると言えます。

## 死にたい気持ちは揺れ動いている

亡くなる直前まで、本当は「生きたい」という気持ちと「死んで楽になりたい」という気持ちの間で揺れ動いていることが多いことが分かっています。決して硬い決意とは言えません。

また、「今日こそ死のう」「今こそ死のう」という強い気持ちは、時間単位でしか続かないことも分かっています。偶然助かった方でも、回復した後には「ああ、生きていてよかった」とおっしゃる方がほとんどです。

## 追い詰められる前に

冷静で最善の道を選ぶような判断ができなくなっていて、傍から見れば「何か生きる道があるんじゃないか」と思える場合でも、「もう自分には死ぬしかないんだ」と思ってしまう状態になります。

だからこそ、そこまで行く前に、誰かに話すこと、相談することがとても大切です。あなたの苦しみを聴いてくれる人は、必ずいます。
