# 依存症家庭で身についた生きづらさ

依存症の家庭で育った人は、大人になってから「生きづらさ」を感じることがあります。それは、子どもの頃に身についた習慣が、今の生活にも影響しているからです。

## 子どもの頃にしていたこと

依存症の家庭では、子どもは自分の気持ちより親の気持ちを優先するようになります。親の機嫌を読み取って行動する、邪魔にならないように静かにしている、みんなを笑わせようとする。

それは「性格」ではなく、その場を生き延びるための戦略だったのです。

## 今の生きづらさにつながること

- 自分の気持ちや欲求が分からない
- 「このままの自分では価値がない」と感じる
- 人に必要とされないと不安になる
- 人間関係で距離感が分からない
- コントロールしたり、されたりする関係になりやすい

## 親のようにはならないと決めていたのに

「絶対に親のようにはならない」と決心していたのに、気づいたら同じようなパターンを繰り返している。そういうこともあります。

それは、あなたが弱いからではありません。子どもの頃に身についた習慣は、意識しないと変えられないものなのです。でも、気づくことができれば、変えていくことができます。
