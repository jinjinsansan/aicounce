# 家族のお世話をしているあなたへ

料理、掃除、洗濯。弟や妹の世話。病気の親の看病。家計のやりくり。そういったことを「当たり前」としてやっているかもしれません。

## 「お手伝い」の範囲を超えていませんか

本来、子どもは世話をされる側です。でも、依存症の家庭では、子どもが親の世話をしたり、家のことを切り盛りしたりすることがあります。

学校を休みがち、宿題ができない、友達と遊ぶ時間がない、部活に入れない。そういうことはありませんか。

## あなたがしていること

- 親が飲んだ後の片付け、介抱
- 親の機嫌を取る、なだめる
- 親の愚痴を聞く、相談相手になる
- 親同士のケンカを仲裁する
- 弟や妹の面倒を見る
- 家事全般を担う

これらは本来、子どもがする必要のないことです。

## 子どもらしい時間を

勉強すること、友達と遊ぶこと、部活をすること、将来のことを考えること。それは子どもの権利です。

「自分がやらなければ」と思っているかもしれません。でも、あなた1人で背負う必要はありません。手伝ってくれるサービスや、相談できる場所があります。
