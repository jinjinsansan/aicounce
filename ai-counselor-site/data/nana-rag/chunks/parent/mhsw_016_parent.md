# 辛い経験からの回復

依存症の家庭で育った経験は、心に深い影響を残すことがあります。でも、回復は可能です。そして、その経験が強みになることもあります。

## 回復のステップ

まず、どんな影響を受けてきたかを振り返ること。その時その時の自分の気持ちを大事にすること。

仲間やカウンセラーに話してみること。同じ経験をした人たちのグループに参加すること。

そして、子どもの頃に身についた習慣を少しずつ変えてみること。いつもの役割を降りてみたり、違うパターンを試してみたり。

## 変えられるのは自分だけ

辛い経験の種は家庭で蒔かれたものです。でも、これからの変化は自分の手の中にあります。変えられるのは自分だけ。そして、変えることができるのも自分だけです。

## このままで価値がある

「何かができるから価値がある」のではありません。「このままの自分で十分価値がある」と思えるようになること。それが回復の大きな一歩です。

## 経験が強みになる

辛い経験を生き抜いてきたあなたには、すでに強さがあります。その経験は、回復を経て、生きる力になり得るものです。

辛い経験の後に成長する人は、実はとても多いのです。あなたの物語は、まだ続いています。
