# 電話相談を使ってみる

「心の健康相談統一ダイヤル」をはじめ、電話で相談できる窓口があります。匿名で、顔を見せずに話せるので、身近な人には言いにくいことも相談しやすいかもしれません。

## どんな相談でも大丈夫

「自殺を考えているわけじゃないんですけど、いいですか」と恐縮しながら電話をかけてくる方もいます。どんな悩みでも大丈夫です。

複雑に重なった問題で行き詰まっている方、身近な人に話せずに1人で抱え込んでいる方、そういった方からの電話が多いそうです。

## 話すことの意味

30分ほどの短い時間で、問題が完全に解決することは難しいかもしれません。でも、話を聴いてもらい、共感してもらい、気持ちを整理する中で、「話せてよかった」「気持ちが楽になった」と感じる方が多いのです。

今日あった出来事を話す、怒りを感じたエピソードを話す。そんな電話も大切な相談です。

## 死にたい気持ちを話していい

「死にたい気持ちを話しても、本当に死ぬ人はこんなところに電話しない」と思うかもしれません。でも実は、話したくても受け止めてもらえなかった経験が重なって、話せなくなってしまう方が多いのです。

だからこそ、今こうして死にたい気持ちを話せていることは、とても大切なことです。電話をかけてくれて、本当によかった。そう思って相談員は話を聴いています。
