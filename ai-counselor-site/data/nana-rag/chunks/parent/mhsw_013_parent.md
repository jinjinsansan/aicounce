# 家で辛いことがあったら

家の中で、叩かれたり、怒鳴られたり、無視されたり、世話をしてもらえなかったり。そういうことがあるなら、それは「あなたが悪い」からではありません。

## これは虐待かもしれない

- 叩かれる、蹴られる、物を投げられる
- ひどい言葉で傷つけられる
- 無視される、存在を否定される
- ご飯を作ってもらえない、世話をしてもらえない
- 親同士の暴力を見せられる

これらは「しつけ」ではありません。子どもが受けてはいけないことです。

## 忘れられている気持ち

親が自分のことで精一杯で、あなたのことを見てくれない。気にかけてもらえない。そういう「忘れられている」感覚は、とても辛いものです。

「自分には価値がない」「助けは来ない」と感じるようになるかもしれません。でも、それは本当のことではありません。あなたには価値があります。助けてくれる人がいます。

## 助けを求めていい

辛いことがあったら、誰かに話してください。学校の先生、スクールカウンセラー、保健室の先生、親戚、電話相談。誰でもいいです。

「家のことを外で話してはいけない」と思っているかもしれません。でも、あなたの安全が一番大切です。助けを求めることは、裏切りではありません。
