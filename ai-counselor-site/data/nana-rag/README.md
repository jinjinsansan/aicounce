## 📁 構成

```
mhsw_rag_system/
├── chunks/           # RAGチャンクファイル
│   ├── parent/       # 親チャンク（詳細説明）
│   └── child/        # 子チャンク（核心メッセージ）
├── docs/             # ドキュメント
│   ├── CHUNK_INDEX.md    # チャンク一覧
│   ├── SYSTEM_PROMPT.md  # AIカウンセラー用プロンプト
│   └── README.md         # 開発ドキュメント
└── README.md         # このファイル
```
