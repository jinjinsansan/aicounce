# 諦めるの本当の意味 - 明らかに眺める

諦めるっていうのは一応何かに挑戦するとか行動するということが前提になってるんだけど、何でもかんでも諦めちゃうっていうのはどうかな。

**諦めちゃうと「明らかに眺める」という意味なんですよ。**

抵抗して抵抗して抵抗してる時っていうのは頭が回転しないんです。

お釈迦様が言った。**今のあなたには今の旦那でちょうどいいね。今のあなたには今の奥さんでちょうどいいね。今のあなたには今の親でちょうどいいんです。**今のあんたには今の会社で本当はちょうどいい。

自分が変わると人の態度まで変わっちゃう。人が変われば全て変わってくる。

足の長い人もいりゃ短い人もいる。**なんとかなるものはなんとかしましょうよ。**でも**なんともならないものってある。**この親より隣の親が良かったって言ってもなんともならない。

**どうにもなることを諦めろって言ってんじゃない。どうにもならないことを諦める。**

明らかに眺めた時、**それでも幸せになる方法を考えなきゃいけないのが修行なんです。**

フランス人になることはできないけど、フランスに行くことはできる。インディアンになることはできないけど、インディアンに会いに行くことはできる。

**できないことは諦めて明らかに眺めると、やれることが分かってくるんです。**
