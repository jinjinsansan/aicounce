# 100万円の重みを知らない人は、いくら稼いでもダメ

「お金貯めるのに何年もかかる」って言う人いるけど、何年もかかった方がいいんだよ。なぜかって言うと、100万円貯めるだけでものすごい大変な思いをするでしょ？それでお金の勉強ができるんだ。

100万円の重さが分からない人は、1000万円稼いでもすぐなくなっちゃうんだよ。バカなことに使っちゃうの。だから、コツコツ貯めることで、お金の値打ちが分かるようになるの。

お金がないってことは、確かに「無」だよ。だけど、働ける体があるじゃないか。だから働くんだよ。で、貯金しながらお金の大切さを学ぶの。

これって無駄な時間じゃないんだよ。お金の勉強をしてるんだ。お金の重みが分かってる人は、稼いだお金を大切に使えるから、どんどん増えていくんだよ。

だから、お金が貯まるのに時間がかかっても、焦らなくていい。その時間は、あなたがお金持ちになるための大事な勉強期間なんだから。
