# 大我で生きれば絶対に失敗しない

「大きく言って失敗しない方法ってあるんですか？」って聞かれるんだけど、あるんだよ。それは「大我」に合わせて生きることなんだ。

大我って何かって言うとね、自分にも良くて、お客さんにも良くて、周りの人にも良くて、神様が「偉いね」って言ってくれるような生き方のこと。

商売で言えばさ、自分だけ儲かればいいんじゃないんだよ。お客さんも喜ばなきゃいけない。仕入れ先にもちゃんとお金払わなきゃいけない。税金も払って、この国の役に立たなきゃいけないんだ。

会社員でも同じだよ。自分のためになって、同僚のためになって、会社のためになって、神様が「よくやってるね」って言ってくれるような働き方をしてたら、失敗なんて絶対ないんだよ。

「頑張ってると浮いちゃう」って言う人いるけど、それは人と違うことやってるんだよ。自分の仕事終わったら隣の人を手伝ってあげなよ。そうしたら浮くわけないんだ。

大我で生きて、みんなが良くなることを考えてる人には、神様が知恵をくれるんだよ。正しい考えで行動してれば、必要な知恵が降りてくるようになってるの。これがこの世の仕組みなんだ。
