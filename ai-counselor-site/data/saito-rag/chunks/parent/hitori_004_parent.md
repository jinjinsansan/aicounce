# いいと思ったことは全部やりなさい - ここは行動の星だから

「何をやったらいいですか？」ってよく聞かれるんだけどね、俺に聞いてもダメなんだよ。俺ができることとあんたができることは違うんだから。

だから、いいと思ったことは全部やるの。全部だよ。なぜかって言うと、この星は「行動の星」だからなんだ。

行動の星に来て行動しないのは間違いなの。東大の試験問題が全部分かっても、書かずに出てきたら0点なんだよ。知ってるだけじゃダメなの。やらなきゃダメなんだ。

「どうやったらうまくいきますか？」って聞く人いるけど、俺がやるとうまくいくことを、あんたがやってもうまくいかないことあるんだよ。同じ「ぽっちゃりして可愛いね」って言っても、喜ぶ人と怒る人がいるでしょ？

だから、自分が「これはうまくいくんじゃないか」と思うことを果敢にやるの。やってみると、うまくいくかいかないかすぐ分かる。うまくいかなかったら改良する。それだけなんだよ。

悩んでる暇とか落ち込んでる暇なんてないからね。出すと同時に「どこが悪いんだろう」って見てなきゃいけないの。これが行動の星の生き方なんだよ。
