# この世は思いと行動 - 秀吉のチャンスの連鎖

あの世は思えばいいの。カツ丼と思えばカツ丼出てくる。念の世界だから。

でも**この世は思いと行動なんだよ。この星はそういう星。**

綺麗になりたいなって思っても、寝癖がついてると寝癖はそのまま。綺麗になりたいと思ったら櫛入れるとかお化粧するとか。赤いドレスが欲しいと言ったら、赤いの買うかこれ赤く染めるかしかない。

**思ってるだけではダメ。思ってるとチャンスが来るの。チャンスに対して行動すればいいだけ。**

秀吉を知ってるよね。最下級のお百姓さん、畑も持ってないからお百姓さんの下だった。

最初に望んだのは侍になりたい。でもいきなり侍にはなれない。なれたのは足軽。しかも信長の所の足軽。殿様自体がいつ殺されるかわからないような潰れかかったところの1番下。

**「足軽か」じゃないの。「あ足軽になれた、よかった」**って一生懸命やると下っぱの侍になるチャンスが来る。

**「なんだ1番下か」じゃなく「ああ侍になれた嬉しい」**→大将になるチャンスが来る。

**「ああ良かった」**→大名になるチャンスが来る。

**「ああ良かった」**→天下を取るチャンスが来る。

**無理はしてないの。良かった良かったと思いながら来たチャンスを掴んでるだけ。**
