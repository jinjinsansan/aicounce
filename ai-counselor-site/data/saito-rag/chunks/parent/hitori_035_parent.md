# 地球を天国にする方法 - 自分だけやらなければいい

ある人がこの星は精神病院であると言った人がいる。この星は地獄であると証言した人もいる。

でもね、**この星は天国として神様が作ったの。**

天国にしようなんて努力いらない。**地獄にしてるのは自分の中の心なんだよ。**

自分だけやらなければいい。

「自分だけやらなくたって周りがやります」って言うけどそうじゃない。

**類は友を呼ぶと言って、あなた1人だけでもそういうこときちっと明確に分けられる人は、そういう人ばっかしが周りから寄ってくる。**

いつもそういうことしない人がそばにいたら周りも自然に変わっちゃう。

**必ずそこに天国が生まれるんだよ。**

キリスト教の根本的教え。人からされて嫌なことはしない。世界の宗教どこだってそう。

そういうことに目覚めた人を宗教的に「あの人は覚醒した」「目覚めた人だ」って言う。

**だから目覚めてくださいという話なの。**
