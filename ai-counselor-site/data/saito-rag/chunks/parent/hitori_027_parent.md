# 悩みの9割は消える - ほったらかしとけばいい

人間は生まれる前に自分の人生決めてくるんだよ。自分に超えられない悩みってない。自分で超えられないような悩みは決めてこないの。

もうダメだって思うけど、**もうダメじゃないんだよ。**

いいこと教えてあげる。

**悩みの9割以上はほったらかしとくと消えちゃう。**

なんとかなるのは悩みじゃないの。何ともならないやつは考えてもならない。

5年前の悩み言ってごらん。覚えてないでしょ？「僕はこうやって解決したんです」というの、ない。**時が解決しちゃう。**

その間に苦しんでるか楽しんでるか。**要は時間だけ。**

すごい悩みのように思うけど、実は大した悩みじゃないの。時間が来ると消えるように、時間が来ると出てくる。どのくらいの時間で消えるか大体決まりがあるんだよ。

**ほったらかしときゃいい。**

今度つけといて、悩みが出たら。見てると勝手になくなってる。1つも解決してないから。

**解決できるやつは悩みじゃないの。**

だから悩んでるとき、「これは時間が解決するんだ」って思えばいい。その間、苦しまないで楽しく過ごしてりゃいいんだよ。
