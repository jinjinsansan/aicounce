# モテる方法 - 男と女の3条件

宇宙のエネルギーとかパイプの話とかいい話なんだけど、ほとんどの人が知りたがってるのは「どうしたら毎日楽しく生きれるだろうか」「どうしたらあの部長が怒らなくなるだろうか」なんだよね。

**宇宙の話よりも、どうしたら自分がモテるようになるかってことの方が気になる。**分かるかい。

じゃあ教えてあげる。

**男がモテる3つの条件：**
1. **優しい**
2. **強い**
3. **手が早い**

優しいだけじゃダメなんだよ。優しくて強くなきゃいけない。この3つを備えてる人は非常に少ないの。

**女がモテる3つの条件：**
1. **可愛い**
2. **優しそうに見える**
3. **手が早い**

「手が早い」って女性の場合はどういうことかって言うとね、狙った男がいたらちょっと余分に見つめるとか、「私もそれ見てみたいわ」とか言うの。**今日あなたに興味があるのよっていう態度を示す。**

昔は親が結婚相手を決めた。待ってりゃ周りが何とかしてくれた。でも今はスピードの時代で人を当てにできないの。

自分に少しもアプローチをしてくれない女に手が早いったら、それ痴漢かストーカーだよ。

**恋するって楽しいんだから、この男好みだなと思ったらちょっと見つめるとかアプローチしないとダメ。**

**愛情というのは息が合わないと、波動が合わないとできないんだよ。**
