# 正しさより楽しさ - 天国に行ける人

幸せって単純ですよ。**楽しいということです。**

どうしたら楽しめるんだろう、この人生をどうやったら楽しめるんだろう。

**人生はどっちが正しいかじゃないですよ。どっちが楽しいかなんです。**

楽しさをうんと追求してください。

「意地悪して楽しいですか」って、**意地悪するより人に親切にした方が楽しいんです。**もう少し深く考えてください。

普通の人ってのは「どっちが正しいか」言い争いっかしてるの。あんた言い争いしてて楽しい？友達を失うだけだよ。理屈って言われるだけだよ。

相手が少しぐらい違ったら「あ、そういう意見もあるんだ、楽しいね」。相当間違ってたこと言ってても「こんなに間違ってても立派に生きてんだ、世の中って楽しいね」。

**今幸せな人しか天国に行けないんですよ。**

正しいことをしてる人が天国行くんじゃない。**楽しく正しいことをしてる人です。楽しくなんです。**

ボランティアやりながら不幸そうな顔してる人いるんです。ボランティアが正しいと思ってんですけど本当はやりたくない。だから顔が歪んでる。本当にやりたかったら顔はニコニコしてる。**そしたら幸せなんです。**
