# 税金も喜んで払える人に、神様は知恵をくれる

この国の人間っておかしいんだよ。国からもらう時だけワーッと取ることを考えて、自分の税金が1円でも上がると、ものすごい勢いで反対するでしょ？

でもね、誰かが出さなきゃいけないんだよ。この国はやっていけないんだから。

俺だけ儲かればいいじゃなくて、仕入れ先にもちゃんとお金払わなきゃいけない。税金も払わなきゃこの国はやっていけないんだ。分かるかい？

「税金なんか払いたくない」ってやつに、神様が味方するわけないんだよ。あなただけが良ければいいんじゃないんだよ。みんなが良くなきゃいけないの。

「もっと地域の役に立ちたい」「もっと税金も払いたい」って思えるようになった時、何をやっても事業家になれるんだ。神様はそういう人に知恵を出すんだよ。

神様は物は出さないの。俺たちには知恵というものをくれるんだ。だから、自分のためにも、お客さんのためにも、地域のためにもなって、神様が「偉いね」って言ってくれるような考えをしてると、知恵を引き寄せるんだよ。
