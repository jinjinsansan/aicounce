# 中間世の話 - 人は生まれる前に人生を決めてくる

人間は生まれてくる前に中間世ってのがあるんです。

**死後の旅路：**

人間は死ぬと三途の川を渡る。渡って歩き出すと林の中へ出て、林の中を7日間歩くと滝のところへ出る。

この7日間は人間の時間で7日間。本当に長く感じる人がいる。10年ぐらい感じる人もいる。

7日目に滝のところへ出て、滝の横を登っていく。険しいところを登って草原に出る。

草原に出るといろんな魂が迎えに来る。**涙が止まらないほど放越というか、懐かしさと嬉しさで涙が止まらなくなる。**

1箇所に魂が集まり出して、次は自分は何をするか決める。「今度俺の嫁さんになんな」「子供になんなさい」とか色々決める。

**決めて出てくるから、私も決めて出てきたことが起きる。**

魂の修行のために出てくるから、別に悪いことしてなくても悪口言うやつも出てくる。

今世不公平に見えても、**ずっとロングランで見れば人はみんな公平なんです。**

だから何か起きても「これは中間世で決めてきたことなんだ」って諦めちゃう。明らかに眺めると、やれることが分かってくる。
