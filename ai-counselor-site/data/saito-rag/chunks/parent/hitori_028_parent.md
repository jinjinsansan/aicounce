# 直感を信じる - 直感ってのは神の領域

中学の先生が「これから国際化するから英語ぐらいできなきゃダメだよ」って言った。英語を見てると「あ、これいらない」ってすぐ分かった。方程式も「あ、これいらない」って。

**英語がいらないって言ってんじゃないの。私の人生ではいらないってことが分かるの。**

方程式はいらないんじゃない。いる人もいる。ただ私には「あ、これいらない」ってわかるんだよ。

アイアムボーイを覚えた。**1回も使わないうちにボーイじゃなくなっちゃった。**

**人間の直感ってすごいの。いらないと本当にいらない。**

初めて誰か紹介されて「こいつ怪しい」と思ったら怪しいんだよ。「こいつ性格悪そう」と思ったら絶対悪い。

「僕はどこの学校出てて、親父はこういう人で、親戚はどこどこの社長で」と言ってるうちに「すごい人に会った」と思っちゃう。**本当にすごい人っていちいち初めて会った人にそんなこと言わないんだよ。**

騙された人に「最初どういう感じした？」と聞くと「なんか怪しいと思った」って言うの。

**直感って騙せないんだよ。直感ってのは神の領域。**

精神論なんかやってると「根はいい人だ」とか言ってるうちに騙されちゃうの。最初の直感を信じなよ。
