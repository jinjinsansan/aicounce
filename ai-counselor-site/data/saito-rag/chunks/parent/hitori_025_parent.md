# 誰でも井戸の中に水はある - 井戸とポンプの話

あのね、じんちゃんは井戸なんだよ。ポンプのパイプに水が入ってるよね。こうやると水の力で下の水が上がってくる。

でも長い間ポンプを使わないと、水が落ちちゃってパイプに空気が入るんだよ。そうするとカチャカチャやっても水は上がってこないの。

その時どうするか。**呼び水つってバケツで水を入れるとパイプがいっぱいになって、下から水が上がってくるんだよ。**

これが人間にも言えるんだ。

誰でも井戸の中に水はあるんだよ。**空の人はいない。**でも長い間使ってないと、カチャカチャやっても出てこないの。ないと思い込んじゃってるんだよね。

だから訪ねてくる人には、気が下がっちゃってる人がいっぱいいる。本当のあなたの良さを出してあげようとしても、カチャカチャやっても出てこない。

そういう時は**三味一体みたいのやってから過去・現在・未来を見る。気を送ってあげてからやる。**バケツ1杯分の水を送ってあげるのと同じなんだよ。

できたら3日間ぐらい続けて受けた方がいい。パイプも汚れてると錆が出たり汚れが出るからね。

だからね、あなたの中には必ず素晴らしいものがあるの。出てこないのは、ずっと使ってなかっただけなんだよ。呼び水入れてあげれば、ちゃんと出てくるからね。
