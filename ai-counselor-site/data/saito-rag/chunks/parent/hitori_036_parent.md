# 積極的に逃げる - うさぎと虎の話

人間って死ぬとあっちの世界で次の人生決めてくるんだよ。本当は乗り越えられないことはない。解決できない問題ってない。

でもみんなが忘れてることがある。そのやり方忘れてるからうまくいかないの。

**逃げるの。積極的に逃げるんだよ。**

逃げちゃいけないと思ってるから苦しくなっちゃう。親も先生も「逃げちゃいけないよ」って言うけど、**自分たちは都合の悪い時逃げまくってんだよ。**

人生って嫌なやつっているんだよ。苦手なやつって絶対いる。それでも立ち向かわなきゃいけないと思って勇気持って立ち向かう。

**うさぎが勇気出して虎に立ち向かったって、向こうから見たら餌が来たと思うだけだよ。**

うさぎは虎に立ち向かうんじゃない。**積極的に、虎がうさぎ見つける前にうさぎが虎見つけるんだよ。だから耳でかいだろ。**

みんな「逃げちゃいけない」と思いながら1歩引くと心に傷がつく。傷つきながらやってって、家まで帰ってきて部屋に入って蓋しめて出られなくなっちゃう。

**引きこもりもうつ病も「逃げちゃいけない」って言ってるからだよ。**

逃げる時はこうだよ（積極的に）。
