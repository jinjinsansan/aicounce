# 失敗は落ち込むためじゃない - 改良するためにあるんだよ

失敗して落ち込んでる人、多いよね。でもね、人は落ち込むために失敗するんじゃないんだよ。改良するために失敗するんだ。

一発で当てようと思ってたの？そんな甘くないよ、世の中は。逆上がりだって一回じゃできないでしょ？何回も練習してできるようになるんだ。仕事も同じなんだよ。

新しいことを始めたら、5回か6回は改良するつもりでいなきゃダメなの。最初から「これは完璧だ」なんて思っちゃいけない。出すと同時に「どこが悪いんだろう」って、じーっと見てなきゃいけないんだ。

社長業って何かって言うとね、「改良業」なんだよ。改良して出す、しばらく見て、また改良して出す。この繰り返しなの。

で、面白いことにさ、最初は10回改良が必要だったのが、次は9回になって、8回になって...だんだん頭の中で改良しながら挑戦できるようになってくるの。そのうち一発で当てられるようになるんだよ。

だから、うまくいかなくても落ち込む必要なんてないの。「あ、ここを改良すればいいんだな」って分かっただけ。それって成功に近づいてるってことなんだよ。
