# 因果の法則 - なぜ自分にこんなことが起きるのか

この世には因果というのがある。原因があるから結果がある。

小さい時お兄ちゃんにいたずらされた。精神的ショック。親にも言ったけど「そんなこと人前で言っちゃダメだよ」と言われた。お兄ちゃんも許せないしお母さんも許せない。

「私は悪くない。ちっちゃい時だから何の原因もなくただ被害者なんだ」と言うけれど...

**その子が人間に生まれること10万回、何回も何回も生まれ変わってた時、前世で男の子だった時に自分の妹にそういうことをしたの。**

だからその因果が今出てきてる。出ると同時にもう消えてなくなった。

それを「お兄ちゃんが悪い、親が悪い」って言ってると、**その因果がもっと大きくなってこの世でもっと悪いことを起こす。**

**因果を消す方法は簡単なんだよ。**

「出た因果は消えたんだ、ありがたい」って思えばいい。

**壺を買う必要もなければご祈祷する必要もない。**薬で因果は消えない。因果を消す薬ってあるんですか？

「俺が前世でやったことなんだよ、ありがたいね」この一言がどんどん消してく。

**どんどん出てきてもいいよ、どんどん消しちゃうから。**
