# 不幸な人は人をいじめる - 幸せにならないといけない理由

人は他の動物と違って悩んだり苦しんだり不幸になると、**人をいじめるという傾向がある。**弱い方へ弱い方へ向く。男の人なら女の人に向いちゃう、女の人は子供に向いちゃう。

いつも「上を目指せ」と言われてきた。学校でも勉強でもスポーツでも。上を上を目指していくと、本当に上になれる人はごくごく稀。その結果いつも自分が苦しくなっちゃう。

**幸せを人と対象にするといつも苦しくなっちゃう。**

幸せは心が決める。自分の心が幸せだなと思ったり「ついてるね」と言ったり。

**幸せだと思わないとあなただけじゃなくて、あなたの中に誰かをいじめる種が入ってるの。**これを出しちゃいけない。出したら自分が鬼になっちゃう。

人は人として生まれた人間が優しくしてあげりゃ仏にもなるし神にもなれる。鬼の部分が出ちゃったら悪魔にも鬼にもなれちゃう。

**人間が自分が幸せだと思ってる時に、人をいじめようとは思わないんだよ。**

必ずつまらない時、面白くない、心が辛い、心が苦しい→人をいじめたくなっちゃう性質がある。

**だから幸せになるって真剣に考えなきゃいけないの。**
