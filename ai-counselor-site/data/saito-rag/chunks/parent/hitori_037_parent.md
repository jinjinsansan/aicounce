# 諦めるの本当の意味 - 明らかに眺める

諦めるっていう言葉あるでしょ。何でもかんでも諦めちゃうっていうのはどうかなって思うよね。

でもね、**諦めるってのは「明らかに眺める」という意味なんだよ。**

抵抗して抵抗して抵抗してる時っていうのは頭が回転しないの。

お釈迦様が言った。今のあなたには今の旦那でちょうどいい。今のあなたには今の奥さんでちょうどいい。今のあなたには今の親でちょうどいい。**今のあんたには今の会社で本当はちょうどいい。**

人が変われば全て変わってくるんだよ。

足の長い人もいりゃ短い人もいる。もうちょっと長ければって言ったって、**なんとかなるものはなんとかしましょうよ。でもなんともならないものってあるの。**

この親より隣の親が良かったって言ってもなんともならない。フランス人の方が良かったのよって言ってもどうにもならない。

**どうにもなることを諦めろって言ってんじゃない。どうにもならないことを諦める。**

明らかに眺めた時、**それでも幸せになる方法を考えなきゃいけないのが修行なんだよ。**

フランス人になることはできないけど、フランスに行くことはできる。**できないことは諦めて明らかに眺めると、やれることが分かってくるの。**
