# 「何もない」と思ってるあなたへ - 無から有を生み出す話

「自分には何もない」って思ってない？地位もない、名誉もない、お金もない...でもね、そう思った瞬間に「夢」が生まれてるんだよ。

「事業家になりたいな」「もっといい人生送りたいな」って思った。それが「有」の始まりなんだよ。何もないと思ってた人間が、夢を持ったんだから。

で、お金がないって言うけどさ、よく考えてみな。働ける体があるじゃないか。手があって、足があって、頭があるんだよ。それって「無」じゃないよね。

世の中ってバラ色なんだよ。だってさ、働きに行けば仕事のやり方を教えてくれた上に、給料までくれるんだよ。ラーメン屋になりたいなら、ラーメン屋で働けばいい。作り方教えてもらって、お金ももらえる。すごいことだよ、これは。

「お金貯めるのに何年もかかる」って言う人いるけど、それでいいんだよ。その間に100万円の重みが分かるようになるんだ。100万円貯めるのに苦労した人は、お金の大切さが身に染みてるから、バカなことに使わないんだよ。

だから、無から有を生み出すって、難しいことじゃないの。夢を持って、体を使って働いて、コツコツ始める。それだけなんだよ。
