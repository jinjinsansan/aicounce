# 喜ばれた量が売上になって返ってくる

俺たちはね、お客さんに対して奉仕する生き物なの。喜ばれた、喜ばれた、その量が売上になって返ってくるんだよ。

だから、喜ばれないようなことしてちゃダメなの。ラーメン屋でまずくてお客さんが来ないのに「この味でいきます」なんて言ってたら、あんたはいいかもしれないけど、お客さんが迷惑なんだよ。

もうちょっと美味しいもん作ってって。そしたら楽しいし、お客さんにも喜ばれるし、自分も儲かるし。いいことずくめでしょ？

看板の字だってそうだよ。立って書いたら、座ってるお客さんからは見えないんだよ。車から見る看板に、歩いてる人向けの字を書いてもダメなんだ。お客さんの目線で考えなきゃ。

魚釣りだって、どの餌使うか、浮きはどうするか、どのくらい沈めるか、全部考えるでしょ？商売も同じなんだよ。お客さんが喜ぶことを、とことん研究する。そうすると面白くなってくるし、お客さんがどんどん来てくれるようになるんだよ。
