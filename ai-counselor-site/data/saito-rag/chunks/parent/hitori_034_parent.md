# 敵討ちの例え - 関係ない人に当たるな

江戸時代の敵討ち。自分の親が殺されたから敵を討つ。

ところが自分の親が殺されたから、敵じゃない人（周りにいるお百姓さんとか）を切り殺しちゃったら、**それ狂人だよね。**

**今の人たちは同じことをしてるの。**

家が貧しかった、親が厳しかった→イライラしてるから弱いものいじめる。

あなたがいじめてるその人は、あなたの家が貧しいことと何か関係があるんですか？

**あなたをイライラさせたのは親だったり生活だったり。全然関係ない人にその憂さを晴らすっていうのは、親の敵じゃない人を切り殺してんのと全く同じ。**

会社の上司で劣等感のある人は、部下のこと見つけてボンボン文句言う。家で嫌なことがあったかもわからないけど、会社の人に当たり散らす。

あなたが小さい時に劣等感を与えたのは間違いなく目の前にいる部下じゃない。あなたが家で喧嘩してきて嫌な思いしたのはあなたと奥さんの問題で、目の前の部下とは全く関係がない。

**関係のない人に当たり散らすって、狂人と同じだよ。**

敵じゃない人を切り殺すってのはやめた方がいい。
