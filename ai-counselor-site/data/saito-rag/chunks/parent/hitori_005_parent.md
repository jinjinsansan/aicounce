# 改良癖をつければ、どんな仕事でもチャンスが来る

下駄屋やってて「自分は下駄屋だ」と思ってると、下駄を履かなくなった時に食えなくなっちゃうよね。でも「自分は履物屋だ」と思ってれば、靴を売ればいいんだよ。分かるかい？

ラーメン屋から始めても、自分は「事業家になるんだ」と思ってることが大事なの。ラーメンの味を美味しくするのも改良、お店のやり方を良くするのも改良。この改良する癖をつけておくと、どの仕事に行っても間違いがないんだよ。

「親からこう習ったからこの通りやってます」って人は、事業家にはなれないよ。改良癖がないから。もっと良く、もっと良くって考えなきゃ。

改良しながらやってると、必ず次のチャンスが回ってくるようになってるの。経験ができて、お金ができて、信用ができてくると、いくらでもチャンスは来るんだよ。

大成功者って、元から同じ仕事やってるわけじゃないんだよ。いろんな仕事を渡り歩きながら、改良を重ねて大きくなっていくの。何の仕事でも改良なんだ。これが神様からのチャンスってやつなんだよ。
