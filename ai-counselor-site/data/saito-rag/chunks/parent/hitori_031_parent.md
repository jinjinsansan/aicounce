# ありがとうと言われる生き方 - スタートラインとゴール

ありがとうって言うといいことあるよ。あるし、いいことなくなります。

言えなかった人がありがとうって言うようになったら周りも神様も喜ぶ。

でもね、寝てた子供がハイしたら親喜ぶよね。でも30過ぎてハイしてたら親が喜ぶんだろうか。

**ありがとうっていうのは当たり前なの。**

ありがとうっていうのはマラソンで言うとスタートラインに立ったようなもん。**これからありがとうって言われる人生になるんだよ。それがゴールなの。**

ほとんどの人はスタートラインで休んでる人とか寝ちゃってる人とかいる。

あの世にいた時、これから地球というとこ行ってきます、日本というとこ行ってきますって神様と約束することがある。

**「そこでありがとうと言われる人生を送ってきます」**

お蕎麦屋さんで誰か通りづらそうだったらちょっと椅子ずらしてあげるとか、細かいことでいいの。

**ありがとうって言われる生き方しようって思いながら生きれば、それだけでうまくいくんだよ。**

幸せじゃない人はずっと「なぜ幸せじゃないか」言ってます。でもその中に「人からありがとうと言われようとして生きてんだ」というとこが1つもない。

**あなたの泣き事聞かされて爽やかにはなれません。**
