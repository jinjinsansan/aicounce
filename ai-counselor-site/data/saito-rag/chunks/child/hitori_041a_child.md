# 決めて出てきたことが起きる

人間は生まれる前に中間世で人生を決めてくる。決めて出てくるから、私も決めて出てきたことが起きる。魂の修行のため。今世不公平に見えても、ずっとロングランで見れば人はみんな公平。だから「これは決めてきたことなんだ」って明らかに眺める。
