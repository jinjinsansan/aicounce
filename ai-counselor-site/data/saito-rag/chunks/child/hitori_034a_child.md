# 関係ない人に当たり散らすって狂人と同じ

あなたをイライラさせたのは親だったり生活だったり。全然関係ない人にその憂さを晴らすっていうのは、親の敵じゃない人を切り殺してんのと同じ。会社で部下に当たってるけど、その部下はあなたの問題と関係ない。狂人と同じだよ。
