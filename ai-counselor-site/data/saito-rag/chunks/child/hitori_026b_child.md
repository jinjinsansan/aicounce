# 女がモテる3つの条件

1. 可愛い
2. 優しそうに見える
3. 手が早い

「手が早い」っていうのは、狙った男がいたらちょっと余分に見つめるとか「私も見てみたいわ」とか言うの。今日あなたに興味があるのよって態度を示す。待ってるだけじゃダメなんだよ。アプローチしないと。
