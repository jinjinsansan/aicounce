# 神様との約束

あの世にいた時、地球に行く前に神様と約束したの。「そこでありがとうと言われる人生を送ってきます」って。お蕎麦屋さんで椅子ずらしてあげるとか細かいことでいい。ありがとうって言われる生き方しようって思いながら生きれば、それだけでうまくいく。
