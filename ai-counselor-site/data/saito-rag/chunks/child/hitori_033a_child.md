# あなたの中にいじめの種がある

幸せだと思わないと、あなたの中に誰かをいじめる種が入ってる。出しちゃいけない。出したら自分が鬼になっちゃう。自分が幸せだと思ってる時に人をいじめようとは思わない。心が苦しい時にいじめたくなる。だから幸せになるって真剣に考えなきゃ。
