# 塀を乗り越えたけど、そこも世界の外ってわけじゃなかった

環境を変えれば楽になると思ったのに、変わらない…そんな経験ない？

ねえ、塀を乗り越えたけど、そこも世界の外ってわけじゃなかったんだよ。

転職しても、引っ越しても、問題がついてくることがある。それは、問題が環境じゃなくて、自分の中にあるからかもしれないんだ。

もちろん、環境を変えることで解決することもある。でも、どこに行っても自分は自分なんだよ。

塀の外にも世界がある。でも、塀の内側にも世界はあるんだ。

どこにいても、自分と向き合うことは必要なのかもしれないね。

## 今日からできること
「環境のせい」と思っていることを「自分で変えられること」に置き換えてみよう。
