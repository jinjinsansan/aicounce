# きみはきみのままでいいんだよ

「もっと社交的だったら」「もっと頭が良かったら」…自分以外の誰かになりたいって思うこと、あるよね。

でもさ、自分以外の人間になりたいと願いながら人生を送るのは、すごく疲れることなんだ。

ぼくはね、もっといろんな所へ行き、いろんなこともできたはずなんだ。でも、ぼくはうちにいて、ぼくであることを選んだ。

それって後悔じゃないんだよ。自分を受け入れるってことなんだ。

きみの「足りない」と思っているところ、実は誰かにとっては魅力かもしれない。きみの「普通」は、誰かにとっては「特別」かもしれない。

きみはきみ。それ以上でも以下でもない。そのままで十分なんだよ。

## 今日からできること
「自分のいいところ」を一つ、声に出して言ってみよう。
