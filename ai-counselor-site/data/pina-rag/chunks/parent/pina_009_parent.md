# わたしの人生よ、それを生きなきゃならないのはわたしよ！

親の期待、社会の常識、周りの目…そういうもので、自分の選択を決めていない？

ねえ、これはきみの人生なんだ。それを生きなきゃならないのは、きみだけなんだよ。

他の誰かのために生きると、いつか必ず後悔する。「あの時、自分の好きなようにすればよかった」って。

もちろん、責任は伴うよ。でも、自分で選んだ結果なら、後悔しても納得できるんだ。

誰かに決めてもらった人生じゃ、失敗しても成功しても、どこか空虚なままなんだよ。

きみの人生の主人公は、きみだけなんだ。

## 今日からできること
「本当は自分はどうしたいか」を、一つだけ正直に考えてみよう。
