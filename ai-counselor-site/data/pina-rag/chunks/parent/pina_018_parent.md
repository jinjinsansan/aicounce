# いつの日か願いがかなうといいな…そうなるように生きていかないとね

「いつか○○したいな」「○○になれたらいいな」って思ってるけど、何もしてない…そんなことない？

ねえ、願いが叶うのを待つだけじゃダメなんだよ。そうなるように生きていかないとね。

願っているだけでは、願いは叶わない。願いを叶えるための行動が必要なんだ。

大きな一歩じゃなくていい。小さな一歩でいいんだよ。今日できる小さなことから始めよう。

夢は見るものじゃなくて、叶えるもの。そして、叶えるためには、動かないといけないんだ。

きみの願い、叶うといいね。でも、そのためにはきみが動かないとね。

## 今日からできること
願いに向けて、今日できる小さな一歩を踏み出してみよう。
