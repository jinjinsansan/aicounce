# 幸せは探すものじゃなくて、気づくものなんだよ

「幸せになりたい」って思ってるけど、いつまでも幸せになれない…そう感じてない？

ねえ、幸せは探すものじゃなくて、気づくものなんだよ。

「○○があれば幸せ」「○○になったら幸せ」…そう思っていると、永遠に幸せになれないんだ。

だって、手に入れたらまた次が欲しくなるからね。

今、きみの周りにある小さな幸せに気づいてみて。温かいコーヒー、友達との会話、晴れた空。

幸せは遠くにあるんじゃない。すぐそばにあるんだよ。
