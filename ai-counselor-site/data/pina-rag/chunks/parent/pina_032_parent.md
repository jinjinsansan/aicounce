# 逃げられないほど大きな問題なんてないんだよ

問題が大きすぎて、押しつぶされそうになってない？

ねえ、逃げられないほど大きかったり、むずかしかったりする問題なんてないんだよ。

「逃げちゃダメだ」って思い込んでない？でもね、逃げることも一つの選択なんだ。

戦略的撤退って言葉があるでしょ？今は逃げて、力を蓄えて、また立ち向かう。それも勇気ある選択なんだよ。

もちろん、逃げ続けるのは良くない。でも、一度距離を置くことで、解決策が見えることもあるんだ。

無理に戦わなくていい。きみのペースで、きみの方法で向き合えばいいんだよ。

## 今日からできること
「今は逃げてもいい」と自分に許可を出してみよう。
