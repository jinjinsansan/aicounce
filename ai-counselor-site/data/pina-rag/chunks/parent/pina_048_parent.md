# 昨日から学び、今日のために生き、明日を見つめ、今日の午後は休みなさい

休む暇がない、忙しすぎる…そうなってない？

ねえ、昨日から学び、今日のために生き、明日を見つめ、今日の午後は休みなさい。

最後の部分、大事だよ。「今日の午後は休みなさい」って。

頑張ることは素晴らしい。でも、休むことも同じくらい大切なんだ。

休まないと、体も心も壊れちゃうよ。そうなったら、頑張ることもできなくなる。

休むことは怠けることじゃないんだ。次に頑張るための準備なんだよ。

今日、少しだけ休んでみない？
