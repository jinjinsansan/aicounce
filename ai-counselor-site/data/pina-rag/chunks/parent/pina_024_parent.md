# 本当に必要な時に来てくれて、側にいてくれる人

困った時、誰かに頼れてる？

ねえ、友情は年月の長さじゃないんだ。友情とは、きみが本当に誰かを必要としたときに、来てくれて、側にいてくれて、そして君の元を去らなかった人のことなんだよ。

長年の付き合いでも、困った時に消える人はいる。逆に、短い付き合いでも、本当に必要な時に駆けつけてくれる人もいる。

大切なのは、年数じゃなくて、深さなんだ。

そして、きみも誰かにとってそういう存在になれたら、素敵だよね。

困った時は、助けを求めていいんだよ。それが友情の始まりになることもあるんだから。

## 今日からできること
困っていることがあったら、一人で抱え込まず誰かに話してみよう。
