# 人を信じるのはいいことなんだよ

過去に裏切られて、人を信じるのが怖くなってない？

ねえ、人を信じるのはいいことなんだよ。ただ、きみはまだ若いだけさ。

若い、というのは年齢のことじゃない。経験を積んで、それでも人を信じる強さを持つってことなんだ。

裏切られたら傷つく。でも、誰も信じないで生きるのは、もっと寂しいんだよ。

信じて裏切られることより、誰も信じられないことの方がつらい。

もちろん、誰でも信じろって言ってるんじゃないよ。でも、信じられる人を見つける努力は、する価値があるんだ。

## 今日からできること
一人だけ、「この人は信じてみよう」と思う人を決めてみよう。
