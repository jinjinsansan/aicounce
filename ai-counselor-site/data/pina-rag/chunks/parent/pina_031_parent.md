# ドン底まで行ったなら、また新しく始められるよ

大きな失敗をして、もうダメだと思ってない？

ねえ、きみがドン底まで行ったんなら、また新しく始められるってことだよ。

ドン底ってことは、これ以上下がないってこと。あとは上がるだけなんだ。

失敗は終わりじゃない。新しいスタートなんだよ。

今まで積み上げてきたものが崩れたように感じるかもしれない。でも、きみの中に残っている経験や学びは、消えないんだ。

ドン底から見上げる空は、案外広いかもしれないよ。さあ、また登り始めよう。
