# 友達はきみのダメなところを全部知っていて、それでも好きでいてくれる人

「本当の友達」って何だろう？

ねえ、友達というものは、きみのダメなところを全部知っていて、それでも好きでいてくれる人のことなんだよ。

SNSのフォロワーが何百人いても、本当の友達は別なんだ。きみの弱さを見せられる人、失敗しても離れない人、それが本当の友達。

たくさんいる必要はないんだよ。一人でもいれば、それで十分なんだ。

そういう人、思い当たる？もしかしたら、きみが「友達」だと思ってない人の中にいるかもしれないよ。

## 今日からできること
「この人には弱さを見せられる」と思う人を一人思い浮かべてみよう。
