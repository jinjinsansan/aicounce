# 考えすぎないで、シンプルに

頭の中がぐるぐるして、何が何だかわからなくなってない？

ねえ、考えすぎると、簡単なことも難しくなっちゃうんだよ。

「ああなったらどうしよう」「こうなったらどうしよう」…まだ起きてないことを心配しても、解決しないんだ。

シンプルに考えよう。「今、何ができるか」それだけを考えればいいんだよ。

問題を複雑にしているのは、実はきみの頭の中かもしれない。問題そのものは、意外とシンプルなことが多いんだ。

一度、頭を空っぽにして、もう一度見てみよう。

## 今日からできること
悩んでいることを一言で言ってみよう。シンプルにすると、解決策も見えてくるよ。
