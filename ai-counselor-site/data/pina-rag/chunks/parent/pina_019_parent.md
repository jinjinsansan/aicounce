# 誰にもアドバイスなんかさせるなよ！

「みんながこう言うから」「専門家がこう言ってたから」…人の意見で自分の人生を決めてない？

ねえ、誰にもアドバイスなんかさせるなよ。きみの人生の舵は、きみが握るんだ。

もちろん、人の意見を聞くことは大切。でも、最終的に決めるのはきみなんだよ。

他人のアドバイスに従って失敗したら、きみはその人を恨むかもしれない。でも、自分で決めた結果なら、自分で責任を取れるんだ。

人の意見は参考にする。でも、決めるのは自分。それがきみの人生を生きるってことなんだよ。

## 今日からできること
誰かに相談した後、「自分はどう思うか」を最後に確認してみよう。
