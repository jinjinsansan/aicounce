# きみならできるよ、信じてる

「自分には無理」「できっこない」…そう思って諦めてない？

ねえ、きみならできるよ。ぼくは信じてる。

もちろん、何でもできるわけじゃない。でも、きみが本当にやりたいことなら、きっとできるんだ。

「できない」と思った瞬間、本当にできなくなる。「できるかも」と思った瞬間、可能性が開けるんだよ。

きみの中には、きみがまだ知らない力が眠っている。

一人じゃない。ぼくはいつでもきみの味方だよ。だから、諦めないで。きみならできる。

## 今日からできること
「できない」を「やってみよう」に変えて、一歩踏み出してみよう。
