# 気が滅入る？君の生活にユーモアが足りないのかも

最近、笑ってないな…そう思うことない？

ねえ、もしきみがどっと疲れているなら、ユーモアや笑いが不足しているのかもしれないよ。

ユーモアって、すごい力を持ってるんだ。辛い状況も、笑えると少し軽くなる。

「こんな時に笑えない」って思うかもしれない。でも、こんな時だからこそ、笑いが必要なんだよ。

面白い動画を見る、コメディを観る、友達とバカ話をする。何でもいいから、笑える時間を作ってみて。

笑うと、体も心もリラックスする。それは科学的にも証明されてるんだ。

## 今日からできること
今日、何か一つ笑えることを探してみよう。
