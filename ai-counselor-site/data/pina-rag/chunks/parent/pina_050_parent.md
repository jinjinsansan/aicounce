# 幸せって、実はシンプルなことの中にあるんだ

幸せって何だろう…わからなくなることってない？

ねえ、幸せって、実はシンプルなことの中にあるんだよ。

美味しいものを食べる。気持ちよく眠る。好きな人と笑う。晴れた日に散歩する。

大げさなことじゃなくていいんだ。日常の小さなことに、幸せは隠れてるんだよ。

「幸せになりたい」と遠くを見るより、「今、幸せだな」と足元を見る方が、幸せに近づけるんだ。

今日、きみの周りにある小さな幸せを探してみて。きっと見つかるよ。

## 今日からできること
今日の中にある「小さな幸せ」を見つけて、味わってみよう。
