# 自分を信じないのがいけないところだよ

「どうせ自分には無理」「きっと失敗する」…そう思ってない？

ねえ、きみのいけない所はね、自分を信じないことなんだよ。自分の能力を信じてないんだ。

失敗するかもしれない。でも、成功するかもしれない。やってみないとわからないんだよ。

自信って、成功体験から生まれると思われがちだけど、実は違うんだ。「やってみよう」と思う勇気から生まれるんだよ。

小さなことでいい。「これならできるかも」ってことを一つやってみる。それが自信の種になるんだ。

きみの中には、きみが思っている以上の力があるんだよ。

## 今日からできること
「できるかも」と思うことを一つ、今日やってみよう。
