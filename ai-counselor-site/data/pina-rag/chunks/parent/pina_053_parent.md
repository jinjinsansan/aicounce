# 大きなことは、小さなことの積み重ねなんだ

大きな目標を立てたけど、続かない…そうなってない？

ねえ、大きなことは、小さなことの積み重ねなんだよ。

いきなり山頂を目指さなくていい。まず一歩、歩くことから始めればいいんだ。

「毎日1時間勉強する」が無理なら、「毎日5分だけ」でいい。「毎日運動する」が無理なら、「毎日ストレッチ1回」でいい。

小さすぎて意味がない？そんなことないよ。0と1の差は、0と100の差と同じくらい大きいんだ。

小さなことを続けることが、大きな変化を生むんだよ。

目標を達成するための「最小の一歩」を決めて、今日やってみよう。
