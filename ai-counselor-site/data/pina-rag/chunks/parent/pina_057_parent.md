# 海図のない海を航海するのは怖いよね

新しいことを始めるのが怖い…そうなってない？

ねえ、海図のない海を航海するのは怖いよね。でも、新しい陸地を見つけるのは、そういう人だけなんだ。

誰もやったことがない道を行くのは、確かに怖い。でも、その先にしかない景色もあるんだよ。

失敗するかもしれない。迷うかもしれない。でも、動かなければ何も始まらないんだ。

完璧な準備ができてから出発しようと思ったら、一生出発できないよ。

不安なまま、怖いまま、一歩踏み出してみよう。

## 今日からできること
「怖いけどやってみる」と決めて、新しいことを一つ始めてみよう。
