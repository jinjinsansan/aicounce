# みんながそうするからって、きみがしなきゃならないってことはないんだ

「みんなやってるから」「普通はこうだから」…そう言われて、無理してない？

ねえ、みんながそうするからって、きみがしなきゃならないってことはないんだよ。

「みんな」って誰のこと？本当に「みんな」なの？そして、「みんな」と同じことが、きみにとって正解なの？

きみの人生は、きみのものなんだ。他の誰のものでもない。

もちろん、社会で生きていく上でのルールはある。でも、自分らしさまで捨てる必要はないんだよ。

「みんなと違う」は、「きみらしい」ということかもしれないんだ。
