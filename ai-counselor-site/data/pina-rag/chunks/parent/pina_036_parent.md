# これは僕にとっていい経験だ

「なんで自分ばっかり」「こんな目に遭うなんて」…そう思ってない？

ねえ、ある意味では、これはきみにとって良い経験なんだよ。色んなことを学べるんだ。

辛い時はそう思えないよね。わかる。でも、後から振り返ると「あの経験があったから」と思えることって、意外と多いんだ。

今すぐそう思えなくてもいい。でも、いつか「あれは良い経験だった」と言える日が来るかもしれない。

その可能性を、心の片隅に置いておいてほしいんだ。

辛い経験は、きみを強くするためにあるのかもしれないよ。

## 今日からできること
辛い経験を「何を学べたか」という視点で見てみよう。
