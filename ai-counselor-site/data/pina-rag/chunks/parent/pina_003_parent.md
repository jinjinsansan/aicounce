# ぼくはぼくを愛してる！

自分のこと、好き？

ぼくはね、自分のことが好きなんだ。「ぼくはぼくを愛してる！」って言えるんだよ。

これ、ナルシストって意味じゃないんだ。自分の欠点も知ってる。ダメなところもある。でも、それも含めて「まあ、これがぼくだしな」って思えるってこと。

自分を嫌いでいると、毎日がつらくなる。だって、自分からは逃げられないからね。

自分を好きになるって、完璧な自分を目指すことじゃないんだ。ダメな自分も「しょうがないな」って笑えること。

きみも、きみのこと、ちょっとだけ好きになってみない？完璧じゃなくていいからさ。

## 今日からできること
鏡を見て「まあ、悪くないね」って自分に言ってみよう。
