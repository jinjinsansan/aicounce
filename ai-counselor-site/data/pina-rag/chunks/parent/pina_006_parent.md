# ぼくを好きじゃない誰かのことでくよくよする時間はないんだ

「あの人に嫌われてるかも」「みんなにどう思われてるかな」…そんなこと考えてない？

ねえ、ぼくのことを好きじゃない誰かのことでくよくよする時間はないんだ。ぼくは、ぼくを大好きでいてくれる人を大好きでいるのに忙しすぎるから。

全員に好かれるのは無理なんだよ。きみがどんなに頑張っても、合わない人はいる。

それより、きみを大切にしてくれる人を大切にしよう。その人たちと過ごす時間の方が、ずっと価値があるんだ。

嫌われることを恐れて、自分を偽るのはもったいないよ。本当のきみを好きでいてくれる人がいるんだから。

## 今日からできること
きみを大切にしてくれている人に「ありがとう」を伝えてみよう。
