# 人生は短いんだから、もっと楽しまなきゃ

毎日がつまらない、楽しいことがない…そう感じてない？

ねえ、人生は短いんだから、もっと楽しまなきゃいけないんだよ。

「楽しむ余裕がない」「楽しいことがない」…本当にそう？

楽しいことは、待っていても来ない。自分で見つけに行くか、作るかしないといけないんだ。

真面目に生きるのも大切だけど、楽しむことも同じくらい大切なんだよ。

今日、何か一つ楽しいことをしてみない？小さなことでいいから。
