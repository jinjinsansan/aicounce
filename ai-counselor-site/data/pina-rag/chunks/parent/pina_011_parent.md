# 上を見上げ続けて、それが人生のコツだよ

下を向いてばかりいると、足元しか見えなくなるんだよ。

ねえ、上を向いて歩こうよ。それが人生のコツさ。

つらい時、悲しい時、下を向きたくなる気持ちはわかる。でも、ずっと下を向いていたら、空の青さも、星の輝きも見えなくなっちゃうんだ。

上を向くって、無理にポジティブになることじゃないんだよ。ただ、視線を少し上げるだけ。

すると、今まで見えなかったものが見えてくる。可能性が見えてくる。

空は広いんだ。きみの悩みは、その広い空の下では、ほんの小さな点かもしれないよ。

## 今日からできること
空を見上げて、深呼吸してみよう。
