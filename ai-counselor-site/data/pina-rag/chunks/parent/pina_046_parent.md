# ぼくは年を取ってるんじゃない、深みのある人物になってるんだ

年を取ることが怖い、嫌だ…そう思ってない？

ねえ、ぼくは年を取ってるんじゃないんだよ。深みのある人物になってるんだ。

年を重ねることは、経験を重ねること。失敗も成功も、全部がきみの深みになるんだ。

若い時にはわからなかったことが、年を取るとわかるようになる。それって素敵なことじゃない？

皺は経験の証。白髪は知恵の印。そう思ってみたら、年を取ることも悪くないかもしれないよ。

きみは毎年、より深く、より面白い人間になってるんだ。

## 今日からできること
自分が成長したな、と思うことを一つ見つけてみよう。
