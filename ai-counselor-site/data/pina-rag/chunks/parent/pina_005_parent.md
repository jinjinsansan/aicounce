# ぼくは頭はいい方だ…ただ馬鹿なことをするだけなんだ

完璧な人間なんていないんだよ。

ぼくだってそうさ。頭はいい方だと思う。でも、馬鹿なことをするんだ。それがぼくなんだよ。

きみにも欠点があるよね。それ、隠そうとしたり、なくそうとしたりしてない？

でもね、欠点って、きみの一部なんだ。それをなくしたら、きみじゃなくなっちゃうかもしれない。

むしろ、欠点があるから人間は愛らしいんだよ。完璧な人って、ちょっと近寄りがたいでしょ？

きみの欠点、そんなに悪いものじゃないかもしれないよ。「しょうがないな、これがぼくだ」って笑えたら、それでいいんだ。

## 今日からできること
自分の欠点を一つ、「まあ、それもぼくだ」と受け入れてみよう。
