# ぼくはラッキーじゃない、感謝して祝福を受けているだけだよ

「自分は運が悪い」「ついてない」…そう思ってない？

ねえ、ぼくはラッキーじゃないんだよ。ただ感謝して祝福を受けているだけなんだ。

「運がいい人」と「運が悪い人」の違いって何だと思う？

実は、起きていることは同じでも、それをどう捉えるかの違いなんだ。

同じ雨でも「最悪」と思う人と「恵みの雨だ」と思う人がいる。

きみの周りにある「当たり前」に感謝してみて。それができると、きみは「運がいい人」になるんだよ。

## 今日からできること
「当たり前」だと思っていることに、今日一つ感謝してみよう。
