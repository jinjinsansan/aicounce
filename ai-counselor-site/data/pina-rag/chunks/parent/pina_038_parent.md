# 見つめていてもご飯皿はいっぱいにならないよ

「いつか良くなる」と待っているだけになってない？

ねえ、見つめていてもご飯皿はいっぱいにならないんだよ。

待っているだけでは、何も変わらない。状況は自分から動かないと、動かないんだ。

もちろん、待つことが必要な時もある。でも、待つだけじゃなくて、待ちながら準備することはできるよね。

チャンスが来た時に動けるように、今できることをしておこう。

願っているだけでは叶わない。行動して初めて、変化は始まるんだ。

## 今日からできること
待っていることがあったら、待ちながらできる準備を一つしてみよう。
