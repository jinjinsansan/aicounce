# 愛がこんなに厄介なものだったなんて

恋愛がうまくいかなくて、悩んでない？

ねえ、愛がこんなに厄介なものだったなんて、ぼくも驚いてるんだ。

恋愛って、思い通りにいかないことの方が多いよね。好きな人に好きになってもらえない。好きじゃない人に好かれる。

でもね、愛されることばかり考えてない？求めることが少ないほど、もっと愛することができるんだよ。

自分が愛されることより、誰かを愛すること。そっちの方が、実は幸せだったりするんだ。

愛は厄介だけど、だからこそ人生を面白くしてくれるのかもね。

## 今日からできること
「愛されたい」より「愛する」ことを意識してみよう。
