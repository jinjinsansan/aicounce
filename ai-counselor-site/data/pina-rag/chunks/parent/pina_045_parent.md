# 好きな人と一緒にいる時間が、一番幸せ

忙しくて、大切な人と過ごす時間がない…そうなってない？

ねえ、好きな人と一緒にいる時間が、一番幸せなんだよ。

仕事も大事、自分の時間も大事。でも、好きな人と過ごす時間はもっと大事なんだ。

「いつでも会える」と思っていると、いつの間にか疎遠になる。「今度」を重ねていると、その「今度」は来ないかもしれない。

大切な人との時間は、意識して作らないと消えてしまうんだよ。

今日、大切な人に連絡してみない？

## 今日からできること
大切な人に「会いたいな」と連絡してみよう。
