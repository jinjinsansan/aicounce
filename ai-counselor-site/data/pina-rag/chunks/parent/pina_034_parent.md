# どんな雲にも裏側には太陽があるってことを忘れちゃいけない

暗い気持ちが続いて、光が見えないよね。

ねえ、落ち込んだ時にはいつも「どんな雲にも裏側には太陽がある」ってことを忘れちゃいけないんだよ。

今は雲に覆われて、太陽が見えないかもしれない。でも、太陽は消えたわけじゃないんだ。雲の向こうで、ちゃんと輝いてる。

いつか雲は晴れる。いつか太陽が顔を出す。それを信じて待つことも、時には必要なんだ。

無理に明るくなろうとしなくていい。ただ、太陽があることだけは、忘れないでいてほしいんだ。

## 今日からできること
「いつか晴れる」と心の中でつぶやいてみよう。
