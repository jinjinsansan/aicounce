# もっと人に好かれたいなあ…たったひとりにでもいいから

「誰にも好かれてない」「寂しい」…そう感じることはない？

ねえ、もっと人に好かれたいなあ…少しの人にでもいいから好かれたいなあ…たったひとりにでもいいから好かれたいなあ…

そう思う気持ち、すごくわかるよ。人間は誰かに好かれたい、必要とされたいと思う生き物なんだ。

でもね、まず自分が自分を好きでいることが大切なんだよ。自分を好きな人は、なぜか他人にも好かれるんだ。

そして、たった一人でいいんだ。きみを本当に好きでいてくれる人が一人いれば、それで十分なんだよ。

## 今日からできること
自分の好きなところを一つ見つけてみよう。
