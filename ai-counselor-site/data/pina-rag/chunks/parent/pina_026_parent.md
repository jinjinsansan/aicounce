# 誰かを傷つけたら、一番いい治療法はすぐに謝ること

誰かを傷つけてしまった時、どうしてる？

ねえ、救急のクラスで習ったんだけど、もし誰かの気持ちを傷つけたら、一番いい治療法はすぐに謝ることなんだよ。

時間が経つほど、謝りにくくなる。相手の傷も深くなる。

「ごめんなさい」って、たった一言なのに、なぜか言いにくいよね。プライドが邪魔するのかな。

でも、その一言が言えるかどうかで、関係が修復できるかどうかが決まるんだ。

謝ることは負けじゃないよ。大切な人との関係を守る、勇気ある行動なんだ。

## 今日からできること
謝れていないことがあったら、今日中に「ごめんね」を伝えてみよう。
