# きみのペースで進めばいいんだよ

周りの人がどんどん先に行って、焦ってない？

ねえ、きみのペースで進めばいいんだよ。

人生は競争じゃないんだ。ゴールも、ルートも、人それぞれ違うんだよ。

誰かが先に行っても、それはその人の人生。きみの人生とは関係ないんだ。

早く進んでも、遅く進んでも、きみの人生はきみのもの。

焦って無理をして倒れるより、自分のペースで確実に進む方が、結局は遠くまで行けるんだよ。

きみは、きみのペースでいいんだ。

## 今日からできること
「人と比べない」と心の中で宣言してみよう。
