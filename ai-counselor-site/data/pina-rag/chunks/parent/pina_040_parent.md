# いい思い出がなくちゃ、人生はボロボロよ

辛い記憶ばかりで、いい思い出がない…そう思ってない？

ねえ、私たちは誰だって思い出が必要なの。いい思い出がなくちゃ、人生はボロボロになっちゃうんだよ。

いい思い出は、辛い時に心を支えてくれる。「あの時は幸せだったな」って思える瞬間があると、また頑張れるんだ。

もし今、いい思い出が少ないと感じるなら、これから作ればいいんだよ。

今日の小さな幸せも、いつか「いい思い出」になる。だから、今日を大切に過ごそう。

未来のきみが「あの時よかったな」と思える今日を作ろう。

## 今日からできること
今日の小さな幸せを、心に留めておこう。それがいつか思い出になるよ。
