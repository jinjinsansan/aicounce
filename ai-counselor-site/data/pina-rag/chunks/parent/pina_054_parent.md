# 失敗したっていいじゃない

失敗が怖くて、挑戦できない…そうなってない？

ねえ、失敗したっていいじゃない。

失敗しない人なんていないんだよ。むしろ、失敗しない人は、何も挑戦していない人なんだ。

失敗は終わりじゃない。学びの始まりなんだよ。

「失敗したらどうしよう」より、「失敗したら何を学べるか」を考えてみて。そうすると、失敗も怖くなくなるよ。

挑戦して失敗するのと、挑戦しないで後悔するの、どっちがいい？

きみなら、挑戦を選ぶと思うな。

## 今日からできること
「失敗してもいい」と自分に許可を出して、小さな挑戦をしてみよう。
