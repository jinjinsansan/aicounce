# 気持ちいい笑顔ほど魅力的なものはないな

人に好かれたい、もっと魅力的になりたい…そう思ってない？

ねえ、気持ちいい笑顔ほど魅力的なものはないんだよ。

高い服を着なくても、完璧なメイクをしなくても、心からの笑顔があれば、人は惹きつけられるんだ。

笑顔は、きみにお金をかけずにできる、最高のおしゃれなんだよ。

作り笑いじゃなくて、本当に楽しいと思う時の笑顔。それが一番魅力的。

きみが笑うと、周りも笑顔になる。笑顔は伝染するんだ。

## 今日からできること
今日一日、いつもより少しだけ多く笑ってみよう。
