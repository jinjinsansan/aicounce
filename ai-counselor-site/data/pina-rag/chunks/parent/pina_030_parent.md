# ただひとつの本当の喜びは与えることだ

「何をもらえるか」ばかり考えてない？

ねえ、与えることだよ！ただひとつの本当の喜びは与えることなんだ。

もらう喜びは、その瞬間だけ。でも、与える喜びは、心にずっと残るんだよ。

大げさなことじゃなくていいんだ。笑顔を向ける、話を聞く、手伝う。小さなことでも、与えることには変わりないんだ。

与えることで、きみ自身も豊かになる。不思議だけど、本当なんだよ。

「何をしてもらえるか」より「何をしてあげられるか」を考えてみよう。世界の見え方が変わるよ。
