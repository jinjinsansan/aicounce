# きみの道は、きみが作るんだよ

人と違う道を歩みたい、でも不安…そう思ってない？

ねえ、きみの道は、きみが作るんだよ。

誰かが歩いた道をたどるのは安全だけど、それはその人の道であって、きみの道じゃない。

きみの道は、きみが一歩一歩歩くことで作られていくんだ。

正解の道なんてないんだよ。きみが選んだ道が、きみにとっての正解になるんだ。

人と違っていい。むしろ、人と違うからこそ、きみの道なんだよ。
