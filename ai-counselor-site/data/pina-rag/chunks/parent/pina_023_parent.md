# きみの涙を理解する一人の友達の方が価値がある

たくさんの知り合いがいても、なんだか孤独…そんなことない？

ねえ、きみの笑顔しか知らない多くの友だちよりも、きみの涙を理解する一人の友だちのほうがはるかに価値があるんだよ。

パーティで楽しく騒げる100人より、つらい時に黙ってそばにいてくれる1人。どっちが大切だと思う？

表面的な付き合いは楽だよ。でも、心は満たされない。

深い友情は、時間がかかる。お互いの弱さを見せ合って、それでも一緒にいることで育つんだ。

きみの涙を見せられる人、いる？
