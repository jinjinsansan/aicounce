# 良い人間になればなるほど、暮らしも良くなる

どう生きればいいのか、わからなくなることってあるよね。

ねえ、良い人間になればなるほど、暮らしも良くなるらしいんだよ。

シンプルだけど、真実なんだ。

人に親切にする。嘘をつかない。約束を守る。そういう「良い人」でいると、周りからも信頼されるし、自分自身も気持ちいいんだ。

難しいことじゃないんだよ。当たり前のことを、当たり前にするだけ。

「どう生きればいいか」の答えは、意外とシンプルなのかもしれないね。
