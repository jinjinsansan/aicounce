# 人生を一冊の本だと考えてごらんよ

人生に行き詰まって、どうしていいかわからない…そんな時ない？

ねえ、新しい人生観が必要なんだよ。きみの人生を一冊の本だと考えてごらんよ。毎日毎日はその本の1ページなんだ。

今日嫌なことがあっても、それは1ページの出来事に過ぎない。本全体を決めるものじゃないんだよ。

そして、次のページはまだ白紙なんだ。何を書くかは、きみ次第。

今までのページが暗くても、これからのページを明るくすることはできる。

きみの人生という本は、まだ終わってないんだよ。

今日という1ページに、何を書きたいか考えてみよう。
