# 完璧じゃなくても、それでいいんだよ

理想の自分と現実の自分、ギャップがあって苦しいよね。

でもさ、完璧な人間なんていないんだよ。いたとしても、そういう人はあまり面白くないしね。

ぼくだって、もっとかっこよくなりたいし、もっとできる犬でいたい。でも、今のぼくが「ぼく」なんだ。

理想を持つのは悪いことじゃない。でも、理想に届かない自分を責め続けるのは、ちょっと酷だと思わない？

今のきみは、今までのきみの人生の結果なんだよ。それを否定したら、きみの人生全部を否定することになっちゃう。

「まあ、こんなもんかな」って、自分を許してあげてもいいんだよ。
