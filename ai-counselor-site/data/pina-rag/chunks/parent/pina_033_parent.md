# 人生の秘訣は、一つの心配事を別の心配事にすり替えること

心配事が次から次へと出てきて、休まらないよね。

ねえ、人生をうまく生きる秘訣は、一つの心配事を別の心配事にすり替えることなんだよ。

…冗談みたいに聞こえる？でも、本当なんだ。

心配は尽きない。一つ解決しても、また別の心配が出てくる。それが人生なんだ。

だから、心配をゼロにしようとしなくていいんだよ。今の心配を乗り越えたら、次の心配に移る。その繰り返し。

心配がある状態が普通なんだ。心配があっても、きみは大丈夫。

## 今日からできること
「心配があって当たり前」と思ってみよう。少し楽になるよ。
