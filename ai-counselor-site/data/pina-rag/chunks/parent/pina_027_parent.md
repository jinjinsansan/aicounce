# 口先ばかりの優しい言葉は何の役にも立たない

「いつでも相談してね」と言いながら、本当に相談したら迷惑そうにする人、いない？

ねえ、口先ばかりの優しい言葉は何の役にも立たないんだよ。

本当の優しさは、言葉じゃなくて行動に現れるんだ。困っている時に黙って手を差し伸べてくれる人、それが本当に優しい人。

言葉だけの人に期待しても、がっかりするだけ。

きみも、誰かに優しくする時は、言葉だけじゃなくて行動で示そう。そうすれば、同じように行動で示してくれる人が、きみの周りに集まってくるよ。

## 今日からできること
言葉だけじゃなく、小さな行動で誰かを助けてみよう。
