# 友達が人生を価値あるものにしてくれる

「一人の方が楽」「友達なんていらない」…そう思うこともあるよね。

でもね、人生という名の旅を価値のあるものにしてくれるのは友達なんだよ。

一人でできることはたくさんある。でも、一人じゃ味わえないものもあるんだ。

喜びを分かち合える人がいると、喜びは倍になる。悲しみを聞いてくれる人がいると、悲しみは半分になる。

友達は、きみの人生に色を添えてくれる存在なんだ。

もちろん、無理に作る必要はない。でも、心を開ける人がいるって、すごくありがたいことなんだよ。

## 今日からできること
友達に「ありがとう」を伝えてみよう。小さなことでいいから。
