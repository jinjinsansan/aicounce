# 配られたカードで勝負するしかないんだよ

ねえ、「あの人みたいだったら」「もっと○○だったら」って思うこと、あるよね。

でもさ、ぼくはこう思うんだ。配られたカードで勝負するしかないのさ…それがどういう意味であれ。

生まれた場所、顔、才能、家庭環境…自分で選べなかったものってたくさんある。それを嘆いても、カードは変わらないんだ。

でもね、どのカードをいつ出すか、それはきみが決められる。同じ手札でも、使い方次第で結果は変わるんだよ。

他の人の手札を羨むより、自分の手札をじっくり見てみない？意外といいカード、持ってるかもしれないよ。

きみの人生は、きみだけのゲームなんだ。

## 今日からできること
自分の「持っているもの」を3つ書き出してみよう。それがきみの手札だよ。
