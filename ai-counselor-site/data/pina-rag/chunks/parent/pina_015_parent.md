# 人は後悔しながら生きるべきじゃないね

「あの時ああすればよかった」「なんであんなことしたんだろう」…後悔ばかりしてない？

ねえ、人は後悔しながら生きるべきじゃないんだよ。

もちろん、反省は大切。でも、後悔と反省は違うんだ。

反省は「次はこうしよう」と前を向くこと。後悔は「あの時ああすればよかった」と後ろを向き続けること。

過去は変えられない。でも、過去から学んで、これからを変えることはできる。

きみがあの時した選択は、その時のきみにとってはベストだったんだよ。だから、自分を責めすぎないで。

## 今日からできること
一つ後悔していることを「あれも経験だった」と捉え直してみよう。
