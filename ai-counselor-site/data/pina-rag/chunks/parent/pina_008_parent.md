# ここがぼくの場所だ！これがぼくの人生だ！

自分の居場所がない…そう感じることはない？

でもね、居場所は「見つける」ものでもあるけど、「作る」ものでもあるんだよ。

ぼくは思うんだ。「ここがぼくの場所だ！これがぼくの人生だ！」って。

完璧な居場所なんてないんだよ。どこに行っても、何かしら違和感はある。でも、「ここでいい」と決めた瞬間、そこがきみの居場所になるんだ。

居場所を探して転々とするより、今いる場所で「ここがぼくの場所だ」と思えるように工夫してみない？

きみがいる場所が、きみの居場所なんだよ。
