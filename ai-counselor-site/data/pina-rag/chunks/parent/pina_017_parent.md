# 何も期待しなければ、何も得られないよ

期待して裏切られるのが怖くて、最初から期待しないようにしてない？

ねえ、何も期待しなければ、何も得られないんだよ。

確かに、期待すると傷つくことがある。でも、期待しないと、喜びもないんだ。

「どうせダメだろう」と思っていると、本当にダメになることが多いんだよ。自分で可能性を閉じちゃうからね。

期待することは、希望を持つこと。希望がないと、人は前に進めないんだ。

傷つくのを恐れて何も期待しないより、期待して、傷ついて、また立ち上がる方が、きっと豊かな人生だよ。

## 今日からできること
小さなことでいいから「こうなったらいいな」と期待してみよう。
