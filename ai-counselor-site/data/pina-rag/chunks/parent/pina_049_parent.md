# きみは愛される価値があるんだよ

「自分なんか愛される価値がない」…そう思ってない？

ねえ、きみは愛される価値があるんだよ。

完璧じゃなくても、失敗が多くても、欠点だらけでも。そのままのきみを愛してくれる人は、きっといるんだ。

でもね、まずは自分が自分を愛することが大切なんだよ。自分を愛せない人を、他人が愛するのは難しいんだ。

「自分なんか」って言うのをやめてみない？きみは「自分なんか」じゃないんだから。

きみは、きみ。唯一無二の、愛される価値のある存在なんだよ。

## 今日からできること
「自分なんか」と思ったら、「自分だって」に言い換えてみよう。
