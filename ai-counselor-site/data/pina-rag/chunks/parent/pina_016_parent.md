# 人生には晴れもあれば雨もある

つらいことが続いて、「いつまでこんな状態が続くんだろう」って思ってない？

ねえ、人生には晴れもあれば雨もある。昼もあれば夜もある。山があれば谷もあるんだよ。

今が谷だとしても、ずっと谷のままじゃない。いつか必ず、また山に登る時が来るんだ。

雨の日が続くと、晴れの日が恋しくなるよね。でも、雨があるから、晴れの日がありがたく感じるんだよ。

どんなことも永遠には続かない。いい時も、悪い時も。

だから、今つらくても、「これも過ぎる」と思っていいんだよ。

## 今日からできること
「これも過ぎる」と心の中でつぶやいてみよう。
