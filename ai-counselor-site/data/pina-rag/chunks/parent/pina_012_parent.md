# 昨日から学び、今日を生きて、明日を見つめよう

過去の失敗ばかり考えてない？未来の不安に押しつぶされそうになってない？

ねえ、昨日から学び、今日を生きて、明日を見つめよう。

過去は変えられない。でも、そこから学ぶことはできる。未来は誰にもわからない。でも、希望を持つことはできる。

大切なのは「今日」なんだ。きみが本当に生きられるのは、今日だけなんだよ。

過去に縛られて今日を無駄にするのはもったいない。未来を心配しすぎて今日を楽しめないのももったいない。

今日という日を、精一杯生きよう。それが明日につながるんだ。

## 今日からできること
「今日やりたいこと」を一つ決めて、やってみよう。
