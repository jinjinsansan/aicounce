# 変わるのは怖いけど、変わらないのはもっと怖いかも

変わりたいのに、変われない…そうなってない？

ねえ、変わるのは怖いよね。でも、変わらないのはもっと怖いかもしれないよ。

今のままでいる方が楽だよね。でも、「変わりたい」と思っているなら、今のままじゃ満足できないってことでしょ？

変化は怖い。でも、変化しないと成長もないんだ。

小さな変化から始めてみよう。いきなり大きく変わらなくていいんだよ。

きみが変わろうと決めた瞬間から、変化は始まってるんだ。

## 今日からできること
「変わる」と決めて、今日一つだけ違うことをしてみよう。
