# 夜中に決めたことは後悔することが多いよ

夜中に一人で考え込んで、どんどん暗くなってない？

ねえ、夜中の3時に思いついたアイディアは、朝の8時に考え直した方がいいんだよ。

夜は心が弱くなるんだ。小さな問題が大きく見えるし、ネガティブな考えが止まらなくなる。

だから、夜中に大事な決断をしちゃダメなんだよ。寝て、朝になってから考え直そう。

朝日の中で見ると、夜中に悩んでいたことが「なんであんなに悩んでたんだろう」ってなることも多いんだ。

今夜は寝よう。明日の朝、また考えればいいんだから。

## 今日からできること
夜中に悩み始めたら「明日考えよう」と決めて寝てみよう。
