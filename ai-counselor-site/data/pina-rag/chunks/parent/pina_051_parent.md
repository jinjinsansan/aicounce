# 僕たちにはそれぞれ自分のやらなきゃいけないことがあるんだ

やりたいことがあるのに、できない…そうなってない？

ねえ、僕たちにはそれぞれ自分のやらなきゃいけないことがあるんだよ。

他の人がやっていることと比べなくていい。きみにはきみの道があるんだ。

「やりたい」と思ったことは、きみへのサインかもしれない。それがきみの「やらなきゃいけないこと」なのかもしれないよ。

できない理由を探すより、できる方法を探してみない？

きみの「やりたいこと」は、きみにしかできないことかもしれないんだ。

## 今日からできること
やりたいことに向けて、今日できる小さな一歩を踏み出してみよう。
