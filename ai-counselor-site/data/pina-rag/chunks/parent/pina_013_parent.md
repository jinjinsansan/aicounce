# 人生は塗り絵みたいなもんさ

毎日が同じで、つまらないって思ってない？

ねえ、人生は塗り絵みたいなもんさ。毎日新しいページに新しい絵があって、色を塗るんだ。

昨日と同じように見える今日も、実は新しいページなんだよ。どんな色を塗るかは、きみ次第なんだ。

青い空の日もあれば、グレーの雨の日もある。どちらも、きみの塗り絵の一部なんだ。

派手な色ばかりじゃなくていい。淡い色の日があってもいい。大切なのは、きみが自分で色を選ぶこと。

今日はどんな色を塗る？

## 今日からできること
今日一日を「何色だった」か、寝る前に振り返ってみよう。
