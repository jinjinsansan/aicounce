# 人生ってソフトクリームみたいなもんさ…なめてかかることを学ばないとね

真面目に頑張りすぎて、疲れてない？

ねえ、人生ってソフトクリームみたいなもんさ…なめてかかることを学ばないとね。

これ、「いい加減にやれ」って意味じゃないんだよ。「もっと気楽に、楽しんで」ってことなんだ。

ソフトクリームを食べるとき、眉間にシワ寄せて必死に食べる人はいないでしょ？楽しんで、味わって食べるよね。

人生も同じなんだ。力を入れすぎると、美味しさを感じられなくなる。

たまには肩の力を抜いて、「まあ、なんとかなるさ」って思ってみない？

## 今日からできること
今日一つだけ、「まあいいか」と力を抜いてみよう。
