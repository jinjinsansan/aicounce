# エペソ人への手紙 2章8-9節

あなたがたの救われたのは、実に、恵みにより、信仰によるのである。それは、あなたがた自身から出たものではなく、神の賜物である。決して行いによるのではない。それは、だれも誇ることがないためなのである。
