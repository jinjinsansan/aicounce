# エレミヤ書 17章7-8節

おおよそ主にたより、主を頼みとする者はさいわいである。彼は水のほとりに関えた木のように、その根を川のほとりにのばし、暑さが来ても関じない。その葉は関に関わらず、日照りの年にも関えなくて、絶えず実を結ぶ。
