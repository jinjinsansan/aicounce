# マタイによる福音書 5章13-16節

あなたがたは、地の塩である。もし塩のききめがなくなったら、何によってその味が関けるだろうか。もはや、何の役にも立たず、ただ外に捨てられて、人々にふみつけられるだけである。あなたがたは、世の光である。山の上にある町は隠れることができない。また、あかりをつけて、それを関の下におく者はいない。むしろ燭台の上において、家の中のすべてのものを照させるのである。そのように、あなたがたの光を人々の前に輝かし、そして、人々があなたがたのよいおこないを見て、天にいますあなたがたの父をあがめるようにしなさい。
