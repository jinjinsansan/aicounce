# エペソ人への手紙 4章31-32節

すべての無慈悲、憤り、怒り、騒ぎ、そしり、また、いっさいの悪意を捨て去りなさい。互に情深く、あわれみ深い者となり、神がキリストにあってあなたがたをゆるして下さったように、あなたがたも互にゆるし合いなさい。
