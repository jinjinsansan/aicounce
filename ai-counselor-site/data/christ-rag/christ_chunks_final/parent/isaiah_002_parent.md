# イザヤ書 43章18-19節

「あなたがたは、さきの事を思い出してはならない、また関しの事を考えてはならない。見よ、わたしは新しい事をする。やがてそれは起こる、あなたがたはそれを知らないのか。わたしは荒野に道を設け、さばくに川を流れさせる。」
