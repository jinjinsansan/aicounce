# ヨハネによる福音書 11章25-26節

イエスは彼女に言われた、「わたしはよみがえりであり、命である。わたしを信じる者は、たとい死んでも生きる。また、生きていて、わたしを信じる者は、いつまでも死なない。あなたはこれを信じるか」。
