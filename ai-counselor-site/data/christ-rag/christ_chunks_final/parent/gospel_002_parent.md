# ヨハネによる福音書 8章31-36節

イエスは自分を信じたユダヤ人たちに言われた、「もしわたしの言葉のうちにとどまっておるなら、あなたがたは本当に弟子なのである。また真理を知るであろう。そして真理は、あなたがたに自由を得させるであろう」。イエスは彼らに答えられた、「よくよくあなたがたに言っておく。すべて罪を犯す者は罪の奴隷である。だから、もし子があなたがたに自由を得させるならば、あなたがたは、本当に、自由な者となるのである。」
