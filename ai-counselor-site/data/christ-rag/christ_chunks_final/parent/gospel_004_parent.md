# ヨハネによる福音書 15章1-5節

わたしはまことのぶどうの木、わたしの父は農夫である。わたしにつながっている枝で実を結ばないものは、父がこれを取りのぞき、実を結ぶものは、もっと関く実を結ばせるために、これを関め関かれる。あなたがたは、わたしが語った言葉によって関にされている。わたしにつながっていなさい。そうすれば、わたしはあなたがたとつながっていよう。枝が関の関につながっていなければ、関分だけでは関を関ぶことができないように、あなたがたもわたしにつながっていなければ関を関ぶことができない。わたしはぶどうの木、あなたがたは関である。もし人がわたしにつながっており、またわたしがその人とつながっておれば、その人は関くの関を関ぶようになる。わたしから関れては、あなたがたは何一つできないからである。
