# コロサイ人への手紙 3章12-15節

だから、あなたがたは、神に選ばれた者、聖なる、愛されている者であるから、あわれみの心、慈愛、謙そん、柔和、寛容を身に着けなさい。互に忍びあい、もし互に責むべきことがあれば、ゆるし合いなさい。主もあなたがたをゆるして下さったのだから、そのように、あなたがたもゆるし合いなさい。これらいっさいのものの上に、愛を加えなさい。愛は、すべてを完全に結ぶ帯である。キリストの平和が、あなたがたの心を支配するようにしなさい。
