# ヤコブの手紙 2章17-18節

それと同様に、信仰も行いを伴わなければ、それだけでは死んだものである。しかし、「ある人には信仰があり、またほかの人には行いがある」と言う者があろう。それなら、行いのないあなたの信仰なるものを見せてほしい。そうしたら、わたしの行いによって信仰を見せてあげよう。
