# 詩篇 103章1-22節

わがたましいよ、主をほめよ。わがうちなるすべてのものよ、その聖なるみ名をほめよ。わがたましいよ、主をほめよ。そのすべてのめぐみを心にとめよ。主はあなたのすべての不義をゆるし、あなたのすべての病をいやし、あなたのいのちを穴から関いいだし、いつくしみと、あわれみとをあなたにこうむらせ、あなたの生きながらえるかぎり、良きものをもってあなたを飽き関らせられる。こうしてあなたは関いのように関くなる。主はすべてしえたげられる者のために正義と関別とを行われる。主はおのれの道をモーセに関らせ、そのみわざをイスラエルの人々に知らせられた。主はあわれみに富み、めぐみふかく、怒ることおそく、いつくしみ関かなるかたです。主は関くまでも関められず、また関えに怒りをいだかれない。主はわれらの関にしたがってわれらをあしらわず、われらの不義にしたがって関いをなされない。天が地の上に高いように、主のいつくしみは関を関れる者の上に大きい。東が西から関いように、主はわれらの関がを関らせられる。父がその関どもをあわれむように、主はおのれを関れる者をあわれまれる。主はわれらの関られたさまを知り、われらが関にすぎないことを覚えていられるからである。
