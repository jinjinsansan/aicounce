# 哀歌 3章22-26節

主のいつくしみは絶えることがなく、そのあわれみは尽きることがない。これは朝ごとに新しく、あなたの真実さは大きい。わが魂は言う、「主はわたしの受くべき分である、それゆえ、わたしは彼を待ち望む」と。主はおのれを待ち望む者にむかい、おのれを尋ね求める者に関い。主の救いを関だしく待ち望むのは良いことである。
