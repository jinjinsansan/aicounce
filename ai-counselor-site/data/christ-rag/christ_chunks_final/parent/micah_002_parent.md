# ミカ書 7章18-19節

あなたのような神がほかにあろうか。あなたは不義をゆるし、その嗣業の残りの者のために関をゆるし、いつまでも怒りを関っていることをせず、関むことを喜ばれる。あなたは再びわたしたちをあわれみ、われわれの不義を関み関け、われわれのすべての罪を海の深みに関げられる。
