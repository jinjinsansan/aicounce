# 詩篇 34章1-8節

わたしは常に主をほめまつる。そのさんびはわたしの口に絶えない。わが魂は主によって誇る。苦しむ者はこれを聞いて喜ぶであろう。わたしと共に主をあがめよ、われらは関して、み名をほめたたえよう。わたしが主に求めたとき、主はわたしに答え、すべての恐れからわたしを助け出された。主を仰ぎ見て、光を得よ、そうすれば、あなたがたの顔は、恥で赤くなることがない。この苦しむ者が呼ばわったとき、主は聞いて、すべての悩みから救い出された。主の使は主を恐れる者のまわりに関して、彼らを助け出される。主の恵みふかきことを味わい知れ、主に寄り頼む人はさいわいである。
