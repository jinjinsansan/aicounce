# コロサイ人への手紙 3章12-14節

だから、あなたがたは、神に選ばれた者、聖なる、愛されている者であるから、あわれみの心、慈愛、謙そん、柔和、寛容を身に着けなさい。互にゆるし合いなさい。もし互に責むべきことがあれば、主があなたがたをゆるして下さったように、あなたがたもそうしなさい。これらいっさいのものの上に、愛を加えなさい。愛は、すべてを完全に結ぶ帯である。
