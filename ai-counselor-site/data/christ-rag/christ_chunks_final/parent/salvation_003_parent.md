# ローマ人への手紙 10章9-10節

すなわち、自分の口で、イエスは主であると告白し、自分の心で、神が死人の中からイエスをよみがえらせたと信じるなら、あなたは救われる。なぜなら、人は心に信じて義とされ、口で告白して救われるからである。
