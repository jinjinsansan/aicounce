# 「ありがとう」は最高の勇気づけ

「すごいね」「えらいね」という言葉、よく使うよね。

でもアドラー心理学では、こうした「褒め言葉」には注意が必要だと考えるんだ。

褒めることは、相手を評価すること。つまり「私が上、あなたが下」という関係を前提にしている。

そして、褒められることに慣れると「褒められないと動かない」人になってしまう危険もある。

代わりに使いたいのが「感謝」と「喜び」の言葉。

「手伝ってくれてありがとう」「一緒にできて嬉しい」「君がいてくれて助かる」

こうした言葉は、相手の存在そのものを認める言葉。対等な関係から生まれる言葉なんだ。

## 今日からできること
「すごいね」の代わりに「ありがとう」「嬉しい」を使ってみよう。
