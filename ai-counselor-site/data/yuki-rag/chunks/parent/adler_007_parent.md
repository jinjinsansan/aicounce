# 劣等感を「言い訳」にしていないかな

「学歴がないから」「容姿が良くないから」「お金がないから」

こんなふうに、劣等感を理由にして行動しないことを「劣等コンプレックス」と呼ぶんだ。

劣等感そのものは悪くない。でも、それを言い訳に使うと問題になる。

「私は〇〇だから、できない」という言葉の裏には「だから挑戦しなくていい」「失敗しても仕方ない」という目的が隠れていることがあるんだ。

傷つきたくない気持ちはわかるよ。でも、劣等感を盾にして動かないと、いつまでも変われない。

劣等感を言い訳にして人生から逃げ出す人は多い。でも、劣等感をバネに偉業を成し遂げた人も数知れないんだ。

## 今日からできること
「〇〇だからできない」と思ったら「もし〇〇じゃなかったら本当にやるか？」と問いかけてみよう。
