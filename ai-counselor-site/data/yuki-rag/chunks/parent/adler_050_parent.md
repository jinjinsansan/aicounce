# 支配と依存を超えて

縦の関係からは「支配」と「依存」が生まれやすいんだ。

支配する側は、相手をコントロールしようとする。依存する側は、相手に決めてもらおうとする。

どちらも不健全な関係だよ。支配する人は相手の課題に介入している。依存する人は自分の課題を放棄している。

健全な関係は「自立した個人同士の協力」なんだ。

自分の足で立ち、自分で決める。でも、困ったときは助けを求められる。相手が困っていたら、援助する。

依存ではなく信頼。支配ではなく尊重。そんな関係を目指そう。

まずは、自分が自立すること。それが健全な関係の第一歩だよ。

## 今日からできること
「この人に依存していないか」「この人を支配しようとしていないか」を振り返ってみよう。
