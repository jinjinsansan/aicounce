# 自分自身の一番の味方になろう

他人を勇気づけることも大切。でも、まず自分を勇気づけることを忘れていないかな？

「どうせ自分なんて」「また失敗した」「もっと頑張らないと」

そんなふうに、自分を責め続けていたら、勇気は湧いてこないよね。

自分を勇気づけるには、まず自分のできていることに目を向けること。小さな進歩を認めること。

失敗しても「これで一つ学んだ」と捉え直すこと。自分に「よく頑張ったね」と言ってあげること。

君は、君自身の一番の味方であるべきなんだ。

自分に厳しくても、勇気は育たない。優しさと厳しさのバランスを取ろう。
