# なぜ「叱る」ではダメなのか

褒めてはいけないなら、叱るのはどうだろう？

実は、叱ることもアドラー心理学では推奨していないんだ。

叱ることは「相手を支配しようとする」行為。恐怖で動かそうとしている。これも縦の関係から生まれるものだよね。

叱られた相手は、表面上は従うかもしれない。でも、心の中では反発している。または、叱られないためだけに行動するようになる。

叱らなくても、伝える方法はあるんだ。

「私はこう感じた」「こうしてくれると助かる」アイメッセージで伝える。相手の課題に介入せず、自分の気持ちを伝える。

対等な関係から、対話で解決しよう。

## 今日からできること
叱りたくなったら「私は〇〇と感じた」というアイメッセージで伝えてみよう。
