# 信用と信頼は、違うもの

「信用」と「信頼」。似ているようで、実は違うんだ。

信用は条件付き。「この人は約束を守るから信じる」「担保があるから信じる」これが信用。

信頼は無条件。「たとえ裏切られる可能性があっても信じる」これが信頼。

銀行がお金を貸すのは「信用」。親が子を信じるのは「信頼」。

アドラー心理学では、深い人間関係を築くには「信頼」が必要だと考えるんだ。

もちろん、無条件に信じれば裏切られることもある。でも、信頼しなければ、本当の関係は始まらない。

裏切るかどうかは相手の課題。信頼するかどうかは君の課題なんだ。

## 今日からできること
大切な人に対して「信用」ではなく「信頼」で接することを意識してみよう。
