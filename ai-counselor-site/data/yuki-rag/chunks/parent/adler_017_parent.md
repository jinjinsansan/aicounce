# 君は、大きなつながりの一部

一人ぼっちだと感じることはないかな？誰ともつながっていないような、そんな孤独感。

アドラー心理学には「共同体感覚」という大切な概念があるんだ。これは「自分は共同体の一部である」という感覚のこと。

家族、友人、職場、地域、社会。君は様々なコミュニティの一員なんだよ。

共同体感覚を持てると、「ここにいていいんだ」という安心感が生まれる。そして、仲間のために何かしたいという気持ちも自然と湧いてくる。

アドラーは言ったんだ。「共同体感覚こそが、幸福への道である」と。

君は一人じゃない。たとえ今は感じられなくても、君を必要としている場所が必ずある。
