# 自由とは、嫌われることを恐れないこと

自由に生きたい。誰もがそう思うよね。

でも、アドラー心理学でいう「自由」は、好き勝手に生きることじゃないんだ。

本当の自由とは「他者の評価を気にしない」こと。他人に嫌われることを恐れないで、自分の信じる道を歩むこと。

他人の目を気にして、自分を偽って生きる。これは不自由な人生だよね。

承認欲求を手放し、課題を分離し、自分の人生を自分で決める。それが本当の自由。

自由には責任が伴う。でも、責任を引き受ける覚悟があれば、君は自由に生きられるんだ。

## 今日からできること
「他人にどう思われるか」ではなく「自分はどう生きたいか」を基準に選択してみよう。
