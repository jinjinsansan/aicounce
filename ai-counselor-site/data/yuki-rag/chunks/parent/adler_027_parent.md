# 助けるのではなく、援助する

困っている人を見ると、つい「助けてあげたい」と思うよね。

でも「助ける」と「援助する」は少し違うんだ。

助けるは、相手の問題を代わりに解決すること。援助するは、相手が自分で解決できるようサポートすること。

アドラー心理学では、相手の力を信じて「援助」することを大切にするよ。

魚を与えるのではなく、魚の釣り方を教える。答えを与えるのではなく、考える手助けをする。

相手の課題に踏み込んで解決するのではなく、相手が自分で乗り越えられるよう、横から支える。

それが本当の援助であり、勇気づけなんだ。

## 今日からできること
誰かを助けたいとき「この人が自分で解決するには何が必要？」と考えてみよう。
