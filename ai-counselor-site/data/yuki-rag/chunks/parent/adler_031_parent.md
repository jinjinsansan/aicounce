# 嫌われることを恐れない

「嫌われたくない」その気持ちが、君を縛っていないかな？

全員に好かれようとすると、誰にも本当の自分を見せられなくなる。みんなに合わせていると、自分が何者かわからなくなる。

アドラーの教えを世に広めた本が『嫌われる勇気』というタイトルなのは、理由があるんだ。

自由に生きるには、嫌われる勇気が必要なんだよ。

もちろん、わざわざ嫌われろということじゃない。自分の信念を貫くとき、結果として嫌う人がいても仕方ないということ。

全員に好かれる人なんていない。君が君らしく生きたとき、離れていく人もいれば、近づいてくる人もいる。

## 今日からできること
「嫌われてもいい」と思えることを一つ決めて、行動してみよう。
