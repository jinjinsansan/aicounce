# 人生を決めるのは、君自身だ

「こんな環境だから仕方ない」「親がこうだったから」と思っていないかな？

アドラーは「人間は自分の人生を描く画家である」と言ったんだ。君を作ったのは君自身。これからの人生を決めるのも君なんだよ。

確かに、生まれた環境や過去の出来事は選べなかった。でも、それにどう対応するかは、今の君が選べるんだ。

「やる気がなくなった」のではなく「やる気をなくす」という決断を自分でしている。「変われない」のではなく「変わらない」という決断を自分でしている。

厳しく聞こえるかもしれないね。でも、これは実は希望の言葉なんだ。自分で決めているなら、自分で決め直すこともできるということだから。

「〜のせいで」と言いそうになったら、「自分は何を選べるか」に言い換えてみよう。
