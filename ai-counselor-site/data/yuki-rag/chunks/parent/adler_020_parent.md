# 誰かの役に立つことで、自分の価値を感じる

「自分には価値がない」そう感じることはないかな？

アドラーはこう言っているんだ。「私は自分に価値があると思えるときにだけ勇気を持てる。そう思えるのは、私の行動が共同体に有益であると思えるときだけである」

つまり、自分の価値を感じるには「誰かの役に立っている」という実感が必要なんだ。

これは自己犠牲とは違うよ。「他者が何をしてくれるか」ではなく「自分が他者に何をできるか」を考えること。

小さなことでいいんだ。挨拶をする、ゴミを拾う、話を聞く。そんな小さな貢献から始めてみよう。

貢献することで、君は「ここにいていい」という居場所を見つけられるよ。

## 今日からできること
今日、誰か一人のために小さな親切をしてみよう。
