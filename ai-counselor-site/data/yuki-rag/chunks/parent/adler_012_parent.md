# 相手のためでも、踏み込んではいけない

「あなたのためを思って言っているのに」こんな言葉を言ったり、言われたりしたことはないかな？

でもね、たとえ相手のためであっても、他人の課題に土足で踏み込んではいけないんだ。

なぜなら、人は自分で決めたことでしか本当には変われないから。押し付けられたアドバイスは、反発を生むだけ。

馬を水飲み場に連れていくことはできる。でも、水を飲ませることはできない。飲むかどうかは馬が決めることなんだ。

君ができるのは、自分の考えを伝えること。そして、相手が助けを求めてきたら手を差し伸べること。

それ以上は、相手の領域。尊重しよう。

アドバイスしたくなったら「これは求められているか？」と確認してみよう。
