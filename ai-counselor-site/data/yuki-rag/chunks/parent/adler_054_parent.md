# 涙にも、理由がある

「泣きたくないのに泣いてしまう」そんな経験はないかな？

アドラー心理学では、泣くことにも目的があると考えるんだ。

たとえば、相手に謝ってほしい。同情してほしい。許してほしい。攻撃をやめてほしい。

涙は、言葉以外のコミュニケーション手段なんだ。

これは「泣くな」ということじゃないよ。泣くことを責める必要もない。

ただ、「この涙で何を伝えようとしているのか」を知ることで、別の表現方法も見つかるかもしれない。

感情を言葉にする。直接伝える。そうすれば、涙に頼らなくても済むかもしれないんだ。

## 今日からできること
泣きそうになったとき「この涙で何を伝えたい？」と自分に聞いてみよう。
