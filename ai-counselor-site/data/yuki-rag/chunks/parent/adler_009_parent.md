# 比べる相手は、昨日の自分だけでいい

SNSを見ると、キラキラした人生を送っている人ばかりに見える。「自分だけが取り残されている」そんな気持ちになることはないかな？

でもね、アドラーはこう言っているんだ。「誰とも競争することなく、ただ前を向いて歩いていけばいい」と。

他人との比較は、終わりがない。上を見ればキリがないし、誰かに勝っても次の比較対象が現れる。

それに、君と他人では環境も経験も才能も違う。比べること自体に意味がないんだ。

比べていいのは「昨日の自分」だけ。少しでも成長していたら、それでいい。

君は君のペースで、君の道を歩めばいいんだよ。

## 今日からできること
比較しそうになったら「昨日の自分より何か成長したか」を考えてみよう。
