# 尊重は、与えることから始まる

「尊重してほしい」そう思っていないかな？

でもね、尊重されたいなら、まず自分から尊重することなんだ。

尊重とは、相手を一人の人間として認めること。相手の考えや感情を大切にすること。意見が違っても、存在を否定しないこと。

アドラー心理学では、全ての人間は対等だと考える。年齢も、立場も、能力も関係なく。

尊重は、相手の課題に踏み込まないことでもある。「あなたはあなたの人生を生きていい」という態度。

まず自分から尊重する。すると、不思議と相手からも尊重されるようになる。

これが「与える」ということなんだ。

## 今日からできること
今日会う全ての人を「尊重すべき一人の人間」として見てみよう。
