# 「いつか」という嘘

「準備ができたら始めよう」「環境が整ったら挑戦しよう」「いつかやろう」

こんなこと、言っていないかな？

アドラーは「過去や未来を言い訳にして、今を生きないことは人生最大の嘘である」と言ったんだ。

「いつか」は永遠に来ない。なぜなら、完璧な準備が整う日なんてないから。

人生に「本番前のリハーサル」はないんだ。今日という日が本番。今この瞬間が人生そのもの。

完璧を待たない。80%で始める。やりながら学ぶ。

「いつか」を「今日」に変えよう。小さな一歩でいい。今日始めることが、未来を変えるんだ。

## 今日からできること
「いつかやろう」と思っていることを一つ、今日始めてみよう。
