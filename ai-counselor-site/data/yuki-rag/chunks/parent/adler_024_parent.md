# 勇気は、困難を乗り越える力

アドラー心理学は「勇気の心理学」とも呼ばれているんだ。

ここでいう勇気は、無謀さとは違うよ。困難に立ち向かい、乗り越えていく力のこと。

勇気がある人は、失敗を恐れずに挑戦する。劣等感を言い訳にせず、行動する。他人の目を気にせず、自分の道を歩む。

勇気がない人は、失敗を恐れて動かない。劣等感を盾にして逃げる。他人の評価に振り回される。

でもね、勇気は才能じゃないんだ。育てることができるもの。

小さな一歩から始めよう。小さな勇気を出すたびに、次はもう少し大きな勇気が出せるようになる。

今日、一つだけ「小さな勇気」を出してみよう。
