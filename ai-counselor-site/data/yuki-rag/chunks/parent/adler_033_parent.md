# ライフスタイル - 君の「人生の地図」

同じような失敗を繰り返してしまう。いつも同じタイプの人に惹かれる。気づくと同じパターンにはまっている。

これは「ライフスタイル」が影響しているんだ。

アドラー心理学でいうライフスタイルは、生活様式のことじゃないよ。人生における思考・行動・感情のパターンのこと。

「自分は〇〇な人間だ」「世界は〇〇な場所だ」「だから〇〇すべきだ」という、人生の地図のようなもの。

これは主に幼少期に形成されて、無意識のうちに君の選択に影響を与えているんだ。

でもね、ライフスタイルは変えられる。まずは自分のパターンに気づくことから始めよう。
