# 勇気づけは、相手の力を信じること

「勇気づけ」とは、困難を克服する活力を相手に与えることなんだ。

でもこれ、褒めることとは違うんだよ。

褒めるのは上から下への評価。「よくできたね」は能力がある人が、ない人にする言葉。

勇気づけは対等な関係から。「ありがとう」「助かったよ」は横の関係から出る言葉。

勇気づけのポイントは、結果よりプロセスを見ること。できたことに注目すること。相手の成長を信じること。

「君ならできる」ではなく「君がやろうとしていることを応援している」という姿勢。

相手の力を信じて、見守る。それが本当の勇気づけなんだ。

## 今日からできること
誰かに「ありがとう」「助かったよ」と伝えてみよう。
