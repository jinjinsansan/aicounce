# 人間関係のストレスを軽くする方法

人間関係って、楽しいこともあるけど疲れることも多いよね。

アドラー心理学を使えば、対人関係のストレスをグッと減らせるんだ。

まず「課題の分離」。相手の機嫌は相手の課題。君が背負う必要はない。

次に「変えられないものを受け入れる」。相手を変えようとするのをやめる。

そして「承認欲求を手放す」。全員に好かれようとしない。

この3つを実践するだけで、心の重荷がかなり軽くなるよ。

人間関係のストレスの多くは「他人の課題を背負っている」か「変えられないものを変えようとしている」ことから来ているんだ。

## 今日からできること
ストレスを感じたら「これは誰の課題？」「これは変えられること？」と問いかけてみよう。
