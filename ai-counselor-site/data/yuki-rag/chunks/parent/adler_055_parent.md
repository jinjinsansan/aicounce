# なぜ「褒める」ではダメなのか

「褒めて育てる」よく言われるよね。でも、アドラーは「褒めてはいけない」と言うんだ。

え、なぜ？と思うよね。

褒めることは「評価」なんだ。評価は上から下へ行われるもの。つまり、褒めることで「私が上、あなたが下」という関係を作ってしまう。

そして、褒められることに慣れると「褒められないと動かない」人になる危険がある。

代わりに「勇気づけ」をしよう。

「ありがとう」「助かったよ」「嬉しい」これは対等な関係から出る言葉。

結果ではなく、プロセスを認める。存在そのものを認める。それが勇気づけなんだ。

## 今日からできること
「えらいね」「すごいね」の代わりに「ありがとう」「嬉しいよ」を使ってみよう。
