# 過去ではなく、目的を見よう

君は「なぜ自分はこうなってしまったのか」と、過去の原因を探していないかな？

アドラー心理学では、人の行動を「原因」ではなく「目的」から理解するんだ。これを「目的論」と呼ぶよ。

たとえば「過去に傷ついたから人を信じられない」と思っているとする。でも、本当にそうかな？実は「人を信じないことで、傷つくリスクを避けたい」という目的があるのかもしれない。

過去の出来事は変えられない。でも、その出来事に「どんな意味を与えるか」は、今の君が決められるんだ。

大切なのは「これからどうするか」だよ。過去に縛られる必要はない。君の人生は、今この瞬間から始まる。

## 今日からできること
過去を振り返りそうになったら、「で、これからどうしたい？」と自分に問いかけてみよう。
