# 小さな勇気を、毎日少しずつ

「勇気を出す」というと、大きな決断を想像するかもしれないね。

でも、勇気は小さなことから始まるんだ。

挨拶をする。「ありがとう」と言う。自分の意見を言ってみる。断ってみる。謝ってみる。

こんな小さなことでも、勇気がいることはあるよね。

大切なのは、小さな勇気を毎日積み重ねること。すると、少しずつ勇気の筋肉が育っていく。

昨日できなかったことが、今日はできるようになる。今日できなくても、明日はできるかもしれない。

焦らなくていい。君のペースで、一歩ずつ進もう。

## 今日からできること
今日一日、「小さな勇気」を一回だけ出してみよう。
