# 交友という人生のタスク

二つ目のライフタスクは「交友」。友人関係のことだね。

仕事と違って、交友には「結果」という明確な目的がない。だから、純粋に「その人と一緒にいたい」という気持ちが大切になる。

交友の関係を通じて、人は他者を理解し、自分を理解する。これを「人間知」と呼ぶよ。

友達の数は関係ない。大切なのは、心を開ける関係があるかどうか。

「この人といると、自分らしくいられる」そんな関係が一つでもあれば、それでいいんだ。

交友は、自分から働きかけることで始まる。待っているだけでは関係は生まれない。勇気を出して、一歩踏み出してみよう。

## 今日からできること
気になる人に、自分から声をかけてみよう。
