# 健全な人と不健全な人の違い

同じ困難に直面しても、人によって対処法は違うよね。

アドラーが考える「健全な人」と「不健全な人」の違いは何だろう？

健全な人は、劣等感を成長のバネにする。不健全な人は、劣等感を言い訳にして動かない。

健全な人は、自分を変えようとする。不健全な人は、相手を変えようとする。

健全な人は、失敗から学ぶ。不健全な人は、失敗を誰かのせいにする。

健全な人は、貢献することで幸せを感じる。不健全な人は、奪うことで幸せを得ようとする。

どちらを選ぶかは、君次第。でも、選ぶことはできるんだ。
