# 「いま、ここ」に全力を注ごう

過去の失敗を悔やんでいる。未来の不安に怯えている。そんな状態で「今」を生きられていないことはないかな？

アドラーは言ったんだ。「人生は連続する刹那である」と。

過去はもう存在しない。未来はまだ存在しない。存在するのは「いま、ここ」だけ。

過去を変えることはできないし、未来を完全に予測することもできない。できるのは、今この瞬間を精一杯生きることだけなんだ。

「いま、ここ」を真剣に生きること。その積み重ねが人生を作る。

目的地を気にしすぎて、今の旅を楽しめないのはもったいない。今日という日を大切にしよう。

## 今日からできること
過去や未来のことが頭に浮かんだら「今、自分にできることは？」に意識を戻そう。
