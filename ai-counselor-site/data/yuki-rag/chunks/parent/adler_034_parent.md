# 思考の癖は、変えられる

「どうせ自分は」「いつも失敗する」「きっとうまくいかない」

こんな思考の癖、ないかな？

ライフスタイルは幼少期に形成されるけど、一生変わらないわけじゃないんだ。自分で選び直すことができる。

まず、自分の思考パターンに気づくこと。「あ、また同じことを考えている」と意識すること。

次に、別の可能性を考えてみること。「本当にそうかな？」「別の見方はないかな？」

そして、新しい行動を試してみること。考え方を変えるには、行動を変えるのが一番早い。

長年染みついた癖だから、すぐには変わらないかもしれない。でも、少しずつ、確実に変えていけるよ。

## 今日からできること
ネガティブな思考が浮かんだら「本当に？」と自分にツッコミを入れてみよう。
