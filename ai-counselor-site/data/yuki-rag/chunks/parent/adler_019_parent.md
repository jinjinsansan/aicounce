# 信頼は、見返りを求めない

「信用」と「信頼」の違いを知っているかな？

信用は条件付き。担保があるから信じる。裏切られたら関係は終わり。

でも信頼は無条件なんだ。裏切られる可能性があっても、それでも信じる。

「そんなの危険じゃない？」と思うかもしれないね。確かに、裏切られることもあるだろう。

でも、裏切るかどうかは相手の課題。君は無条件に信頼することで、深い関係を築く可能性を開くんだ。

警戒ばかりしていたら、誰とも本当のつながりは持てない。

他者を信頼する勇気は、自己受容から生まれる。自分を信じられる人だけが、他人を信じられるんだよ。

今日会う誰か一人を「信頼してみる」と決めて接してみよう。
