# 自立できる人を育てる

子育てや教育のゴールは何だろう？

アドラー心理学では「自立」が大きな目標だと考えるんだ。自分で考え、自分で決め、自分で行動できる人を育てること。

そのためには、手を出しすぎないことが大切。

失敗しそうでも見守る。自分で考える時間を与える。助けを求められたら援助する。

課題の分離を忘れないで。子どもの課題は子どものもの。親が代わりにやってしまうと、成長の機会を奪ってしまうんだ。

「この子のため」と思って先回りすることが、実は自立を妨げていることもある。

信じて、任せて、見守る。それが本当の愛情なんだよ。
