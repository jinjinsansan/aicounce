# それは、誰の課題だろう？

人間関係で悩んでいるとき、一つ大切な問いがあるんだ。「それは誰の課題か？」

アドラー心理学では「課題の分離」という考え方を大切にしているよ。自分の課題と、他人の課題を分けて考えるんだ。

見分け方はシンプル。「その選択の結果を最終的に引き受けるのは誰か」を考えればいい。

たとえば「子どもが勉強しない」という悩み。勉強するかしないかを決め、その結果を引き受けるのは子ども自身。だから、これは子どもの課題なんだ。

他人の課題に踏み込むと、対立が生まれる。自分の課題を放置すると、人生が停滞する。

まずは「これは誰の課題か」を明確にすることから始めよう。

悩みを感じたら「この結果を引き受けるのは誰？」と問いかけてみよう。
