# 「ここにいていいんだ」という感覚

人が抱える最も根源的な欲求は「所属感」だと、アドラーは考えたんだ。

「ここにいていいんだ」「この仲間の一員なんだ」という感覚。これが満たされないと、人は様々な問題行動を起こしてしまう。

注目を集めようとする。反抗する。無気力になる。これらは「居場所がほしい」という叫びなんだ。

じゃあ、どうすれば所属感を得られるのか。

答えは「貢献」だよ。仲間のために何かをすることで、自分はこの共同体に必要な存在なんだと実感できる。

待っているだけでは居場所はできない。自分から関わり、貢献することで、居場所は作られるんだ。
