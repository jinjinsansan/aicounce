# 「変わらない」という決断をしている

変わりたいのに変われない。その苦しさ、わかるよ。

でもアドラーは、ちょっと厳しいことを言うんだ。「変われないのではなく、変わらないという決断を自分でしている」と。

なぜ変わらないことを選ぶのか。それは、今の状態にもメリットがあるから。

変わらなければ、失敗するリスクを避けられる。今の自分を否定しなくていい。新しいことに挑戦しなくていい。

つまり「楽」なんだ。変化は怖いし、エネルギーがいるからね。

でも、変わらないことで失っているものもある。それを見つめる勇気を持とう。

変わることを選ぶのも、変わらないことを選ぶのも、君の自由。でも、選んでいるのは君自身だよ。

## 今日からできること
「変われない」と思ったら「変わらないことで何を得ている？」と問いかけてみよう。
