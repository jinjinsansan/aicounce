# すべての悩みは、対人関係につながっている

仕事の悩み、お金の悩み、将来の不安。いろんな悩みがあるよね。

でもアドラーは言ったんだ。「人生において対人関係以外の問題はないように見える」と。

お金の悩みも、突き詰めると「人にどう思われるか」「家族を養えるか」という対人関係の問題。仕事の悩みも、上司や同僚との関係が絡んでいることが多いよね。

だから、悩みを解決するヒントは、いつも対人関係の中にあるんだ。

「この悩みは、誰との関係から生まれているのか」を考えてみて。そこに答えがあるよ。

人間関係は難しい。でも、だからこそ学びがあり、成長がある。僕たちは一人では生きられないからね。

## 今日からできること
悩みを感じたら「これは誰との関係の問題だろう」と考えてみよう。
