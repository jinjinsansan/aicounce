# 愛という人生のタスク

三つ目のライフタスクは「愛」。これが最も難しいと言われているんだ。

なぜ難しいか。愛の関係は最も近い距離にあるから。距離が近いほど、課題の分離が難しくなる。

アドラー心理学でいう理想の愛の関係とは「一緒にいても自由に振る舞える」こと。

劣等感を抱くことも、優越性を誇示することもない。ありのままの自分でいられる関係。

「二人で一人」になるのではなく「二人でいても、それぞれが一人」でいられること。

そんな関係を築くには、まず自分自身が自立していることが大切なんだ。依存ではなく、対等な協力関係。

## 今日からできること
パートナーとの関係で「課題の分離」ができているか振り返ってみよう。
