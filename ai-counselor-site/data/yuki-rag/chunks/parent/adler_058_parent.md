# 今日から、一歩ずつ

アドラー心理学、いろいろ学んだね。でも「知っている」と「できる」は違うんだ。

大切なのは実践すること。今日から、小さく始めてみよう。

まずは一つだけ選んで。「課題の分離」でも「勇気づけ」でも「いま、ここ」でもいい。

完璧にやろうとしなくていい。うまくいかなくても大丈夫。また明日やればいい。

アドラーは言ったんだ。「人生を変えるのに、魔法は必要ない。日々の小さな選択の積み重ねが人生を作る」と。

知識を増やすより、一つを実践する方がずっと価値があるよ。

さあ、今日から何を始める？
