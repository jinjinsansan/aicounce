# 劣等感は、君を成長させるエネルギー

「自分は人より劣っている」そう感じることはないかな？

実はね、アドラーは劣等感を否定していないんだ。むしろ「劣等感があるのは当然」と言っているよ。

なぜなら、劣等感は「こうなりたい」という理想があるから生まれるもの。目標がなければ、劣等感なんて感じないんだ。

大切なのは、劣等感をどう使うか。

劣等感をバネにして努力する人がいる。一方で、劣等感を言い訳にして動かない人もいる。

同じ劣等感でも、使い方で結果は正反対になるんだよ。

劣等感は敵じゃない。君を成長させようとしてくれている味方なんだ。
