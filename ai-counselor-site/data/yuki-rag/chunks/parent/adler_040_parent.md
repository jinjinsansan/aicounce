# 幸福は、貢献感の中にある

「幸せになりたい」誰もがそう思うよね。でも、幸せって何だろう？

アドラー心理学では、幸福は「貢献感」と深く結びついているんだ。

「私は誰かの役に立っている」「私はこの共同体に貢献している」

そう感じられたとき、人は幸せを感じる。自分の価値を実感できるから。

お金や地位や名誉を得ても、貢献感がなければ心は満たされない。逆に、何も持っていなくても、貢献感があれば幸せを感じられる。

大切なのは、実際に感謝されることじゃないんだ。「誰かの役に立っている」という主観的な感覚。

そのためにまず、自分にできる小さな貢献から始めてみよう。

## 今日からできること
