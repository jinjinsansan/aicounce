# 認められなくても、君の価値は変わらない

「認めてほしい」「褒められたい」「評価されたい」

この気持ち、誰にでもあるよね。でもアドラーは、承認欲求について厳しいことを言っているんだ。

「他者から承認されることを求めてはいけない」

なぜか。承認を求めて生きると、他人の期待に合わせた人生になってしまうから。

褒められるために行動する。認められないと価値がないと感じる。これでは自由に生きられない。

君の価値は、他人が決めるものじゃないんだ。君が決めるもの。

承認されなくても、君には価値がある。その確信を持てたとき、本当の自由が手に入るよ。

## 今日からできること
「認められるため」ではなく「自分がやりたいから」という理由で何かをしてみよう。
