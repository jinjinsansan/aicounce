# 劣等感という名のエネルギー

アドラー自身、子どもの頃はとても病弱だったんだ。くる病を患い、二度も事故に遭った。

でも彼は、その劣等感をバネにして医師を目指し、心理学の巨匠になった。

これを「補償」と呼ぶよ。劣っている部分を埋め合わせようとする力が、成長のエネルギーになるんだ。

劣等感を感じたとき、二つの道がある。

一つは、劣等感を言い訳にして動かない道。もう一つは、劣等感をバネにして努力する道。

どちらを選ぶかは、君次第。でも僕は、君には後者を選ぶ力があると信じているよ。

「あの弱点があったから、今の自分がある」いつか、そう思える日が来る。
