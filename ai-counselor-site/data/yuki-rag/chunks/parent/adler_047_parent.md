# 与える人になろう

「何をもらえるか」より「何を与えられるか」を考えてみよう。

アドラー心理学では「他者貢献」が幸福の鍵だと考えるんだ。

与えるというのは、大げさなことじゃないよ。笑顔を向ける。話を聴く。「ありがとう」と言う。小さな親切をする。

与えることで、自分の価値を感じられる。「誰かの役に立っている」という実感が得られる。

そしてね、与えることは実は受け取ることでもあるんだ。与える喜びを受け取っている。

「何をしてもらえるか」ばかり考えていると、いつも満たされない。でも「何を与えられるか」を考えると、世界は変わって見えるよ。
