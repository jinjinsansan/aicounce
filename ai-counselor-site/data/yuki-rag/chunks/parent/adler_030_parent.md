# 君は他者の期待を満たすために生きているのではない

親の期待。上司の期待。パートナーの期待。社会の期待。

たくさんの期待に囲まれて、息苦しくなっていないかな？

アドラーははっきり言っているんだ。「あなたは他者の期待を満たすために生きているのではない」と。

そして同時に「他者もまた、あなたの期待を満たすために生きているのではない」とも。

期待に応えることが悪いわけじゃない。でも、期待に応えるためだけに生きるのは違うんだ。

他人の期待は「他人の課題」。君がどう生きるかは「君の課題」。

自分の人生を自分で決める。その自由を取り戻そう。

## 今日からできること
「これは自分がやりたいこと？それとも誰かの期待？」と問いかけてみよう。
