# 縦の関係から横の関係へ

「縦の関係」は上下がある関係。親と子、上司と部下、先輩と後輩。

縦の関係では、上の人が下の人を「褒める」「叱る」「評価する」。権力が働くんだ。

「横の関係」は対等な関係。立場は違っても、人間として同じ価値があるという意識。

横の関係では、「感謝する」「尊敬する」「協力する」が中心になる。

アドラー心理学は、人間関係をできるだけ「横の関係」にすることを勧めているんだ。

もちろん、会社などでは役割の違いはある。でも、人間としての価値は対等。

相手を一人の人間として尊重する。その意識を持つだけで、関係は変わるよ。

## 今日からできること
誰かを「上」か「下」で見ていないか、振り返ってみよう。
