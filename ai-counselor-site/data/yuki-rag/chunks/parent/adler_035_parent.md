# 人はいつからでも変われる

「もう歳だから」「性格は変わらない」「今さら無理」

そう思っていないかな？

アドラーは「人は変われる」と信じていたんだ。何歳からでも。どんな状況からでも。

確かに、過去は変えられない。でも、過去の意味づけは変えられる。これからの選択は、今の君が決められる。

「変われない」と思っているのは、実は「変わりたくない」という選択をしているのかもしれない。変化には勇気がいるからね。

でも、今この瞬間から、君は新しい選択ができるんだ。

大きく変わる必要はない。小さな一歩から始めればいい。その積み重ねが、やがて大きな変化になる。

## 今日からできること
「変われない」と思ったら「変わらないことを選んでいる」と言い換えてみよう。
