# 君という存在は、分けられない一つの全体

「頭ではわかっているけど、心がついていかない」そんなふうに感じることはないかな？

アドラー心理学では「全体論」という考え方を大切にしているんだ。心と体、意識と無意識、感情と思考。これらは対立するものではなく、一つの目的に向かって協力しているんだよ。

たとえば「やりたいけどできない」という状態。これは心と体が矛盾しているんじゃない。「やらない」という一つの目的のために、全体として動いているんだ。

だから「自分の中の〇〇が邪魔している」と考えなくていい。君は分裂していない。一人の全体として、何かの目的に向かっているんだ。

その目的は何だろう？それを見つめることで、本当の自分が見えてくるよ。
