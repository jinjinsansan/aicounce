# 境界線は、関係を守るためにある

誰かに頼まれると断れない。自分のことより相手を優先してしまう。そんなことはないかな？

課題の分離は「冷たい人になれ」ということじゃないんだ。むしろ、適切な境界線を引くことで、健全な関係を築けるようになるんだよ。

差し伸べれば手が届く。でも相手の領域には踏み込まない。そんな適度な距離を保つことが大切なんだ。

境界線がないと、君は疲弊する。そして疲弊した君は、本当に大切な人を助けることもできなくなる。

「No」と言うことは、自分を大切にすること。そして、自分を大切にできる人だけが、本当の意味で他人を大切にできるんだよ。
