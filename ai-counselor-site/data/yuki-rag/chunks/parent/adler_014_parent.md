# 変えられるものと、変えられないもの

上司の機嫌。天気。過去の出来事。他人の性格。

これらは、君がいくら頑張っても変えられないものだよね。

でも、変えられるものもある。自分の行動。自分の考え方。自分の言葉。これからの選択。

アドラー心理学では「変えられないものを受け入れ、変えられるものを変えていく勇気を持つ」ことが大切だと教えているんだ。

変えられないものに怒っても、エネルギーの無駄遣い。その力を、変えられることに注ごう。

コントロールできないことで悩むのをやめると、驚くほど心が軽くなるよ。
