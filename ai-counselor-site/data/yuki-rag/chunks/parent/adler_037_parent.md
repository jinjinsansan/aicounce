# 仕事という人生のタスク

アドラーは、人生には3つの重要な課題があると言ったんだ。仕事、交友、愛。これを「ライフタスク」と呼ぶよ。

まず「仕事のタスク」。これは生きていくために避けられない課題だね。

仕事の対人関係は、実は3つの中で最もハードルが低いと言われているんだ。なぜなら、成果という共通の目的があるから。

でも、だからこそ難しい面もある。嫌いな人とも協力しなければならないこともあるよね。

大切なのは、仕事を「自分の貢献の場」として捉えること。給料をもらうためだけじゃなく、誰かの役に立っているという実感を持つこと。

そうすれば、仕事は苦役ではなく、自分の居場所になるんだ。

## 今日からできること
「この仕事は誰の役に立っているか」を具体的に考えてみよう。
