# 「普通」でいる勇気

「特別な存在になりたい」「人と違う何かがほしい」

そう思うことはないかな？

でもアドラーは「普通であることの勇気」を持つことが大切だと言っているんだ。

特別であろうとすると、常に他者との比較の中で生きることになる。勝ち続けなければならない。

普通の自分を受け入れられないのは、実は自己受容ができていないということなんだ。

「普通」は「劣っている」という意味じゃない。特別な存在であろうとする競争から降りて、自分らしく生きること。

君は、特別じゃなくても価値がある。何者かにならなくても、今の君のままで十分なんだよ。

## 今日からできること
「特別になりたい」と思ったら「普通の自分でOK」と言い聞かせてみよう。
