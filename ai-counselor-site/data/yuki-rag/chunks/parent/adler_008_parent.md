# 自慢の裏側にあるもの

自分がいかにすごいかを語り続ける人。他人を見下す人。過去の栄光にしがみつく人。

実はこれ、「優越コンプレックス」と呼ばれる状態なんだ。

優越コンプレックスは、強い劣等感の裏返し。本当は自分に自信がないから、優れているフリをして補おうとしているんだよ。

本当に自信がある人は、わざわざ自慢しない。静かに自分の道を歩むものなんだ。

もし君が誰かの自慢話にイライラしたら、「この人は劣等感と戦っているんだな」と思ってみて。少し優しくなれるかもしれない。

そして、君自身が虚勢を張っていることに気づいたら。それは劣等感を受け入れるチャンスだよ。
