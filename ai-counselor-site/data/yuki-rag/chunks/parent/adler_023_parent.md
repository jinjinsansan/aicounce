# 他者は敵ではなく、仲間

「あいつには負けたくない」「周りはみんなライバルだ」

そんなふうに、人を敵として見ていないかな？

アドラー心理学では、他者を「仲間」として見ることを大切にするんだ。

全員が敵の世界で生きるのはつらい。でも、全員が仲間の世界で生きるなら、どんな困難も乗り越えられる気がしない？

競争ではなく、協力。勝ち負けではなく、共に成長する。

もちろん、全ての人と仲良くなる必要はないよ。でも、基本姿勢として「この人も自分と同じように悩み、頑張っている仲間なんだ」と思えるかどうか。

その視点が、人生を大きく変えるんだ。

## 今日からできること
苦手な人に対して「この人も何かと戦っている仲間だ」と考えてみよう。
