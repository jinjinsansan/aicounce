# 怒りは、目的のための道具

「つい怒ってしまった」「感情が抑えられなかった」

こんなこと、あるよね。でもアドラー心理学では、怒りの見方が少し違うんだ。

怒りは「湧いてくる」ものではなく「使う」もの。つまり、目的があって怒りを選んでいるんだ。

たとえば、部下を叱るとき。「相手を屈服させたい」「言うことを聞かせたい」という目的のために、怒りという道具を使っている。

電話が鳴って急に冷静になれるのは、怒りが抑えられないものではない証拠だよ。

「なぜ怒っているか」より「この怒りで何を得ようとしているか」を考えてみよう。

目的がわかれば、怒り以外の方法も見つかるはずだよ。
