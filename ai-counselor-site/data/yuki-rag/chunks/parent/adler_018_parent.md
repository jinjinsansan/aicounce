# まず、今の自分を受け入れよう

「こんな自分じゃダメだ」「もっと〇〇でなければ」

自分を責め続けていないかな？

共同体感覚の土台は「自己受容」なんだ。これは「自分を好きになる」こととは少し違う。

自己受容とは、変えられない自分を受け入れること。そして、変えられる自分は変えていく勇気を持つこと。

完璧な人間なんていない。欠点があるのが人間なんだ。

「60点の自分でも、まあいいか」そう思えることが、自己受容の始まり。

自分を受け入れられると、他人も受け入れられるようになる。まずは君自身に優しくしよう。

## 今日からできること
「今日の自分、まあまあだったな」と、一日の終わりに自分を認めてあげよう。
