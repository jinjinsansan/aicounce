# 君は、君の課題に集中すればいい

「あの人に嫌われているかも」「こんなことしたらどう思われるだろう」

他人の評価が気になって、動けなくなることはないかな？

でもね、他人が君をどう思うかは「他人の課題」なんだ。君がコントロールできることじゃない。

君ができるのは「自分が正しいと思うことをする」こと。それだけ。

相手が理解してくれるかどうか、評価してくれるかどうかは、相手の課題。

自分の課題に全力を注ぎ、他人の課題は相手に任せる。そうすれば、心はずっと軽くなるよ。

他人の目を気にして生きるのは、他人の人生を生きているのと同じなんだ。

## 今日からできること
「どう思われるか」より「自分はどうしたいか」を優先してみよう。
