# 変えられるのは、自分だけ

「あの人が変わってくれたら」「環境が違えば」「あの出来事さえなければ」

そう思いたくなる気持ちはわかるよ。でも、他人も過去も環境も、君にはコントロールできないんだ。

唯一コントロールできるのは、自分自身。自分の考え方、行動、選択。

アドラーは言ったよ。「健全な人は相手を変えようとせず、自分が変わる」と。

自分が変わると、見える世界が変わる。自分が変わると、相手の態度も変わることがある。

自分を変えるのは怖い。でも、それが人生を変える唯一の方法なんだ。

今日からできる小さな変化から始めよう。
