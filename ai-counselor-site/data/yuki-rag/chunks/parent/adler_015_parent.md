# 相手を変えることはできない

「あの人がもっと〇〇だったら」「どうして変わってくれないんだろう」

そう思うこと、あるよね。でもね、残念ながら他人を変えることはできないんだ。

アドラーはこう言っているよ。「健全な人は相手を変えようとせず、自分が変わる。不健全な人は相手を操作し、変えようとする」

相手を変えようとするほど、関係は悪化する。なぜなら、それは相手の課題に踏み込んでいるから。

でも、君が変われば、相手との関係性は変わる可能性がある。君の態度が変われば、相手の反応も変わるかもしれない。

変えられるのは、いつも自分だけ。でも、自分が変わることで、世界の見え方は変わるんだ。

## 今日からできること
「相手に変わってほしい」と思ったら「自分は何を変えられる？」と考えてみよう。
