# 君が見ている世界は、君が作っている

同じ出来事でも、人によって受け取り方は全然違うよね。

アドラー心理学では「人は現実そのものを見ているのではなく、自分が意味づけした現実を見ている」と考えるんだ。

コップに水が半分。「まだ半分ある」と見るか「もう半分しかない」と見るか。どちらも同じ事実だけど、意味づけが違う。

ピンク色のメガネをかけている人は、世界がピンク色だと思い込んでいる。自分がメガネをかけていることに気づいていないんだ。

君のメガネは何色だろう？「自分はダメだ」「人は信用できない」そんなメガネをかけていないかな？

メガネを外すことはできないけど、かけ替えることはできるよ。
