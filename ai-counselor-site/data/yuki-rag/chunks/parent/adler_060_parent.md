# 人生に意味を与えるのは、君自身

「人生の意味って何だろう？」

アドラーの答えはシンプルなんだ。「人生に意味はない。意味を与えるのは自分自身だ」

これは虚しい話じゃないよ。むしろ希望の話。

人生に決まった意味がないからこそ、君は自由に意味を創れるんだ。

そして、人生は刹那の連続。過去でも未来でもなく、今この瞬間の積み重ね。

「いま、ここ」を精一杯生きること。誰かの役に立つこと。自分らしくあること。

それが、君だけの人生の意味になる。

君の人生は、君が創る物語。どんな物語にするかは、君次第だよ。

## 今日からできること
「今日という日に、どんな意味を与えたい？」と朝、自分に問いかけてみよう。
